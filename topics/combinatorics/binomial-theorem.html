Introduction to the Binomial Theorem

The binomial theorem provides a formula for expanding expressions of the form (x + y)ⁿ, where n is a non-negative integer. This theorem is a cornerstone of algebra and combinatorics, with applications ranging from probability theory to computer science.

Binomial Expansion

The binomial theorem states that for any non-negative integer n:

(x + y)ⁿ = Σk=0n (n k) xn-k yk

= (n 0) xn y0 + (n 1) xn-1 y1 + (n 2) xn-2 y2 + ... + (n n) x0 yn

where (n k) is the binomial coefficient, representing the number of ways to choose k items from n distinct items, calculated as:

(n k) = C(n,k) = n! / [k! × (n-k)!]

Example: Expanding (x + y)⁴

Using the binomial theorem:

(x + y)⁴ = Σk=04 (4 k) x4-k yk

= (4 0) x⁴y⁰ + (4 1) x³y¹ + (4 2) x²y² + (4 3) x¹y³ + (4 4) x⁰y⁴

Calculate each binomial coefficient:

(4 0) = 4!/(0! × 4!) = 1

(4 1) = 4!/(1! × 3!) = 4

(4 2) = 4!/(2! × 2!) = 6

(4 3) = 4!/(3! × 1!) = 4

(4 4) = 4!/(4! × 0!) = 1

Therefore:

(x + y)⁴ = 1·x⁴ + 4·x³y + 6·x²y² + 4·xy³ + 1·y⁴

= x⁴ + 4x³y + 6x²y² + 4xy³ + y⁴

This expansion can be used to calculate the probability of getting different combinations of heads and tails when tossing a coin 4 times.

Binomial Coefficients

Properties of Binomial Coefficients

Binomial coefficients have several important properties that make them useful in combinatorial problems:

  • Symmetry: (n k) = (n n-k)
  • Edge cases: (n 0) = (n n) = 1
  • Pascal's Identity: (n k) = (n-1 k-1) + (n-1 k)
  • Sum of coefficients: Σk=0n (n k) = 2ⁿ
  • Alternating sum: Σk=0n (-1)ᵏ(n k) = 0
  • Vandermonde's Identity: Σk=0r (m k)(n r-k) = (m+n r)

Pascal's Triangle

Pascal's triangle is a triangular array of binomial coefficients, where each number is the sum of the two numbers directly above it.

Row 0: 1

Row 1: 1 1

Row 2: 1 2 1

Row 3: 1 3 3 1

Row 4: 1 4 6 4 1

Row 5: 1 5 10 10 5 1

...

The entries in the n-th row of Pascal's triangle are precisely the coefficients of (x + y)ⁿ.

Example: Using Pascal's Triangle

The coefficients of (x + y)⁵ can be read directly from the 5th row of Pascal's triangle:

Row 5: 1 5 10 10 5 1

Therefore:

(x + y)⁵ = 1·x⁵ + 5·x⁴y + 10·x³y² + 10·x²y³ + 5·xy⁴ + 1·y⁵

This can be used to calculate the probability of different combinations of successful and unsuccessful shots in an archery competition with 5 rounds.

Binomial Theorem: Advanced Applications

Finding a Specific Term

The (k+1)th term in the expansion of (x + y)ⁿ is:

Tk+1 = (n k)xn-kyk

Example: Finding a Specific Term

Find the coefficient of x⁸y⁵ in the expansion of (2x - 3y)¹³, which corresponds to the term where there are 8 factors of x and 5 factors of y in the polynomial.

First, rewrite (2x - 3y)¹³ as (2x + (-3y))¹³.

For a term with x⁸y⁵, we need n-k=8 and k=5 to get xn-kyk.

This means n=13, k=5.

Using the formula for the term:

T6 = (13 5)(2x)⁸(-3y)⁵

= (13 5)(2⁸)(-3)⁵(x⁸)(y⁵)

= (13 5)(2⁸)(-3)⁵x⁸y⁵

Calculate (13 5) = 13!/(5! × 8!) = 1,287

Calculate 2⁸ = 256

Calculate (-3)⁵ = -243

Therefore, the coefficient is 1,287 × 256 × (-243) = -79,964,928

The Binomial Theorem for Special Cases

Some special cases of the binomial theorem have interesting applications:

  • (x + 1)ⁿ: Σk=0n (n k)xk
  • (x - 1)ⁿ: Σk=0n (n k)xn-k(-1)k
  • (1 + x)ⁿ for |x| < 1 and n ∈ ℝ: Σk=0∞ (n k)xk
Example: Special Case Application

Find the sum of the first 6 rows in Pascal's triangle, which can be used to calculate the total number of defeated Mongolian wrestlers in a traditional Mongolian wrestling competition with up to 6 rounds.

We know that the sum of binomial coefficients in row n is 2ⁿ.

Therefore, the sum of the first 6 rows (rows 0 to 5) is:

2⁰ + 2¹ + 2² + 2³ + 2⁴ + 2⁵ = 1 + 2 + 4 + 8 + 16 + 32 = 63

We can verify this equals 2⁶ - 1 = 64 - 1 = 63.

Multinomial Theorem

The multinomial theorem is a generalization of the binomial theorem to expressions with more than two terms.

Multinomial Expansion

For a positive integer n:

(x₁ + x₂ + ... + xₘ)ⁿ = Σ (n; k₁, k₂, ..., kₘ) x₁k₁ x₂k₂ ... xₘkₘ

where the sum is over all non-negative integer values of k₁, k₂, ..., kₘ such that k₁ + k₂ + ... + kₘ = n, and the multinomial coefficient is:

(n; k₁, k₂, ..., kₘ) = n! / (k₁! × k₂! × ... × kₘ!)

Example: Multinomial Expansion

Find the expansion of (x + y + z)³, which represents the probability of different combinations of outcomes in a traditional Mongolian three-man wrestling competition.

Using the multinomial theorem with n=3, m=3:

(x + y + z)³ = Σ (3; k₁, k₂, k₃) xk₁ yk₂ zk₃

where k₁ + k₂ + k₃ = 3.

The possible combinations of (k₁, k₂, k₃) are:

(3,0,0), (0,3,0), (0,0,3), (2,1,0), (2,0,1), (1,2,0), (0,2,1), (1,0,2), (0,1,2), (1,1,1)

Calculate each term:

(3; 3,0,0) = 3!/(3!×0!×0!) = 1 → x³

(3; 0,3,0) = 3!/(0!×3!×0!) = 1 → y³

(3; 0,0,3) = 3!/(0!×0!×3!) = 1 → z³

(3; 2,1,0) = 3!/(2!×1!×0!) = 3 → 3x²y

(3; 2,0,1) = 3!/(2!×0!×1!) = 3 → 3x²z

(3; 1,2,0) = 3!/(1!×2!×0!) = 3 → 3xy²

(3; 0,2,1) = 3!/(0!×2!×1!) = 3 → 3y²z

(3; 1,0,2) = 3!/(1!×0!×2!) = 3 → 3xz²

(3; 0,1,2) = 3!/(0!×1!×2!) = 3 → 3yz²

(3; 1,1,1) = 3!/(1!×1!×1!) = 6 → 6xyz

Therefore:

(x + y + z)³ = x³ + y³ + z³ + 3x²y + 3x²z + 3xy² + 3y²z + 3xz² + 3yz² + 6xyz

Applications in Probability and Statistics

Probability Distributions

The binomial theorem is fundamental in probability theory, particularly for the binomial distribution, which models the number of successes in a fixed number of independent trials.

P(X = k) = (n k) pk (1-p)n-k

where n is the number of trials, k is the number of successes, and p is the probability of success in a single trial.

Example: Binomial Probability

A coin is tossed 6 times. What is the probability of getting exactly 4 heads?

This follows a binomial distribution with n = 6, k = 4, and p = 0.5 (probability of heads on a single toss).

P(X = 4) = (6 4) (0.5)⁴ (1-0.5)² = (6 4) (0.5)⁴ (0.5)²

(6 4) = 6!/(4! × 2!) = 15

P(X = 4) = 15 × (0.5)⁴ × (0.5)² = 15 × (0.5)⁶ = 15 × (1/64) = 15/64 = 0.234375

Therefore, the probability of getting exactly 4 heads is approximately 0.234 or 23.4%.

Statistical Applications

The binomial theorem has numerous applications in statistics, including:

  • Computing moments of probability distributions
  • Deriving sampling distributions
  • Calculating confidence intervals
  • Hypothesis testing
