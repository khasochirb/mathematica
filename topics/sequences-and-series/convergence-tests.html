Convergence Tests

Techniques for determining whether infinite series converge or diverge

Home » Resources » Mathematics Roadmap » Sequences and Series » Convergence Tests

Introduction to Convergence Tests

Determining whether an infinite series converges or diverges is a fundamental problem in calculus. While some series have well-known behaviors, such as geometric series or p-series, many others require specialized tests.

In this lesson, we'll study various tests for convergence that help us analyze different types of series. These tests provide methods to determine convergence without calculating the exact sum of the series.

Before applying any convergence test, remember the necessary condition for convergence: if a series Σaₙ converges, then lim(n→∞) aₙ = 0. Therefore, if lim(n→∞) aₙ ≠ 0 or the limit doesn't exist, the series diverges. However, if lim(n→∞) aₙ = 0, the series may still diverge, so further testing is required.

Comparison Tests

Direct Comparison Test

The direct comparison test compares a series with another series whose convergence behavior is known.

For series with non-negative terms:

  • If 0 ≤ aₙ ≤ bₙ for all n ≥ N (some positive integer) and Σbₙ converges, then Σaₙ also converges.
  • If 0 ≤ bₙ ≤ aₙ for all n ≥ N and Σbₙ diverges, then Σaₙ also diverges.
Example 1: Direct Comparison Test

Determine whether the series Σ(n=1 to ∞) 1/(n² + 2) converges or diverges.

Since n² ≤ n² + 2 for all n ≥ 1, we have 1/(n² + 2) ≤ 1/n² for all n ≥ 1.

The series Σ(1/n²) is a p-series with p = 2 > 1, so it converges.

Therefore, by the direct comparison test, the series Σ(1/(n² + 2)) also converges.

Limit Comparison Test

The limit comparison test is useful when direct comparison is difficult. For series with positive terms:

Consider Σaₙ and Σbₙ with aₙ > 0 and bₙ > 0 for all n ≥ N. Let:

L = lim(n→∞) aₙ/bₙ
  • If 0 < L < ∞ (i.e., L is finite and positive), then Σaₙ and Σbₙ either both converge or both diverge.
  • If L = 0 and Σbₙ converges, then Σaₙ also converges.
  • If L = ∞ and Σbₙ diverges, then Σaₙ also diverges.
Example 2: Limit Comparison Test

Determine whether the series Σ(n=1 to ∞) (3n²+1)/(2n⁴+5) converges or diverges.

Let's compare with bₙ = 1/n², which is a convergent p-series (p = 2 > 1).

L = lim(n→∞) [(3n²+1)/(2n⁴+5)] / (1/n²) = lim(n→∞) (3n²+1)n²/(2n⁴+5) = lim(n→∞) (3n⁴+n²)/(2n⁴+5)

As n → ∞, the highest power dominates, so L = 3/2, which is finite and positive.

Since Σ(1/n²) converges and 0 < L < ∞, the series Σ[(3n²+1)/(2n⁴+5)] also converges.

Integral Test

The integral test applies to positive, continuous, and decreasing functions. If aₙ = f(n) where f is a positive, continuous, decreasing function for x ≥ N, then:

  • If ∫(N to ∞) f(x) dx converges, then Σ(n=N to ∞) aₙ converges.
  • If ∫(N to ∞) f(x) dx diverges, then Σ(n=N to ∞) aₙ diverges.
Example 3: Integral Test

Determine whether the series Σ(n=1 to ∞) 1/(n ln n) converges or diverges.

Let f(x) = 1/(x ln x). This function is positive, continuous, and decreasing for x > 1.

Evaluate the integral ∫(2 to ∞) 1/(x ln x) dx:

Let u = ln x, then du = 1/x dx, so dx = x du.

∫(2 to ∞) 1/(x ln x) dx = ∫(ln 2 to ∞) 1/(xu) · x du = ∫(ln 2 to ∞) 1/u du = [ln|u|](ln 2 to ∞) = ln|ln x|(2 to ∞)

= lim(t→∞) [ln(ln t) - ln(ln 2)] = ∞ - ln(ln 2) = ∞

Since the integral diverges, the series Σ(n=2 to ∞) 1/(n ln n) also diverges.

Therefore, the series Σ(n=1 to ∞) 1/(n ln n) diverges.

Ratio and Root Tests

Ratio Test

The ratio test is especially useful for series involving factorials or exponentials. For a series Σaₙ with aₙ > 0:

Calculate:

ρ = lim(n→∞) |a(n+1)/aₙ|
  • If ρ < 1, the series converges absolutely.
  • If ρ > 1 or ρ = ∞, the series diverges.
  • If ρ = 1, the test is inconclusive.
Example 4: Ratio Test

Determine whether the series Σ(n=1 to ∞) n²/2ⁿ converges or diverges.

ρ = lim(n→∞) |a(n+1)/aₙ| = lim(n→∞) |(n+1)²/2^(n+1)| / |n²/2ⁿ|

= lim(n→∞) (n+1)²·2ⁿ/(n²·2^(n+1)) = lim(n→∞) (n+1)²/(n²·2) = lim(n→∞) (1+1/n)²/2

= 1/2 < 1

Since ρ < 1, the series converges absolutely.

Root Test

The root test is particularly useful for series where the terms involve powers. For a series Σaₙ:

Calculate:

ρ = lim(n→∞) |aₙ|^(1/n)
  • If ρ < 1, the series converges absolutely.
  • If ρ > 1 or ρ = ∞, the series diverges.
  • If ρ = 1, the test is inconclusive.
Example 5: Root Test

Determine whether the series Σ(n=1 to ∞) (2n/n!)^n converges or diverges.

ρ = lim(n→∞) |(2n/n!)^n|^(1/n) = lim(n→∞) |2n/n!|

Using Stirling's approximation for large n: n! ~ √(2πn)(n/e)ⁿ

ρ = lim(n→∞) 2n/[√(2πn)(n/e)ⁿ] = lim(n→∞) 2n/[√(2πn)n^n·e^(-n)]

= lim(n→∞) 2n·e^n/[√(2πn)n^n] = lim(n→∞) 2e^n/[√(2πn)n^(n-1)]

As n → ∞, this approaches 0, so ρ = 0 < 1.

Therefore, the series converges absolutely.

Alternating Series Test

The alternating series test (also known as Leibniz's test) applies to series of the form Σ(−1)ⁿaₙ or Σ(−1)^(n+1)aₙ, where aₙ > 0.

If:

  1. aₙ > a(n+1) for all n ≥ N (i.e., {aₙ} is decreasing), and
  2. lim(n→∞) aₙ = 0

Then the alternating series converges.

Example 6: Alternating Series Test

Determine whether the series Σ(n=1 to ∞) (−1)^(n+1)/n² converges or diverges.

Here, aₙ = 1/n².

1. The sequence {1/n²} is decreasing for n ≥ 1.

2. lim(n→∞) 1/n² = 0.

Therefore, by the alternating series test, the series converges.

Note: The series also converges absolutely since Σ(1/n²) converges.

The alternating series test only tells us about convergence, not absolute convergence. A series may converge conditionally (when the series converges but the series of absolute values diverges).

Absolute and Conditional Convergence

Definitions

For a series Σaₙ:

  • Absolute Convergence: The series Σ|aₙ| converges.
  • Conditional Convergence: The series Σaₙ converges, but Σ|aₙ| diverges.

If a series converges absolutely, it also converges. However, a convergent series may not converge absolutely.

Example 7: Absolute vs. Conditional Convergence

Consider the alternating harmonic series Σ(n=1 to ∞) (−1)^(n+1)/n = 1 - 1/2 + 1/3 - 1/4 + ...

The series of absolute values is Σ(n=1 to ∞) |(-1)^(n+1)/n| = Σ(n=1 to ∞) 1/n, which is the harmonic series and diverges.

However, the alternating harmonic series satisfies the conditions of the alternating series test:

1. The sequence {1/n} is decreasing for n ≥ 1.

2. lim(n→∞) 1/n = 0.

Therefore, the alternating harmonic series converges, but not absolutely. It is conditionally convergent.

Absolute convergence is a stronger property than conditional convergence. Absolutely convergent series are well-behaved—their terms can be rearranged without affecting the sum.

Choosing the Right Test

Selecting the appropriate convergence test depends on the characteristics of the series. Here are some guidelines:

Type of Series Recommended Test
Series with factorials or exponentials Ratio Test
Series with terms raised to powers Root Test
Series comparable to p-series or geometric series Comparison Tests
Series with terms that are rational functions of n Limit Comparison Test with appropriate p-series
Series with terms defined by continuous, decreasing functions Integral Test
Series with alternating signs Alternating Series Test

Often, multiple tests may apply to a given series. It's usually best to choose the test that makes the calculation simplest.

Example 8: Choosing the Right Test

Determine whether the series Σ(n=1 to ∞) n/(3n²+1) converges or diverges.

This series involves a rational function of n. We could use the limit comparison test with a p-series.

As n gets large, n/(3n²+1) ≈ n/(3n²) = 1/(3n), which resembles the harmonic series 1/n with p = 1.

Let's apply the limit comparison test with bₙ = 1/n:

L = lim(n→∞) [n/(3n²+1)] / (1/n) = lim(n→∞) n²/(3n²+1) = 1/3

Since L is finite and positive, and Σ(1/n) diverges, the given series also diverges.
