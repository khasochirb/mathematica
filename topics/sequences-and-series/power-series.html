Power Series

Understanding series representations of functions and their properties

Introduction to Power Series

A power series is a series of the form:

Σ(n=0 to ∞) aₙ(x - c)ⁿ = a₀ + a₁(x - c) + a₂(x - c)² + a₃(x - c)³ + ...

where:

  • x is a variable
  • c is a constant (the center of the series)
  • aₙ are the coefficients of the series

When c = 0, we have a Maclaurin series:

Σ(n=0 to ∞) aₙxⁿ = a₀ + a₁x + a₂x² + a₃x³ + ...

Power series are important because they allow us to represent functions as infinite polynomials, which can be useful for approximating function values, performing integration, or solving differential equations.

Radius and Interval of Convergence

For a power series Σ aₙ(x - c)ⁿ, the radius of convergence R determines where the series converges:

  • For |x - c| < R, the series converges absolutely
  • For |x - c| > R, the series diverges
  • For |x - c| = R, the series may converge or diverge, depending on the specific series

The interval of convergence is the set of x-values for which the series converges. It may be of the form:

  • (c - R, c + R) if the series diverges at both endpoints
  • [c - R, c + R) if the series converges at x = c - R but diverges at x = c + R
  • (c - R, c + R] if the series diverges at x = c - R but converges at x = c + R
  • [c - R, c + R] if the series converges at both endpoints

The radius of convergence can be found using the ratio test or the root test:

R = 1 / lim(n→∞) |a(n+1)/aₙ| (Ratio Test)
R = 1 / lim(n→∞) |aₙ|^(1/n) (Root Test)
Example 1: Finding the Radius and Interval of Convergence

Find the radius and interval of convergence for the power series Σ(n=1 to ∞) xⁿ/n².

Using the ratio test:

lim(n→∞) |a(n+1)/aₙ| = lim(n→∞) |x^(n+1)/(n+1)²| / |xⁿ/n²| = lim(n→∞) |x| · n²/(n+1)² = |x| · lim(n→∞) (n/(n+1))² = |x|

The series converges when |x| < 1 and diverges when |x| > 1.

At x = 1, the series becomes Σ 1/n², which is a convergent p-series (p = 2 > 1).

At x = -1, the series becomes Σ (-1)ⁿ/n², which also converges by the alternating series test.

Therefore, the radius of convergence is R = 1, and the interval of convergence is [-1, 1].

Operations on Power Series

Within their radii of convergence, power series can be treated like polynomials for algebraic operations:

Addition and Subtraction

If Σ aₙxⁿ and Σ bₙxⁿ have radii of convergence R₁ and R₂ respectively, then:

Σ aₙxⁿ ± Σ bₙxⁿ = Σ (aₙ ± bₙ)xⁿ

with radius of convergence at least min(R₁, R₂).

Multiplication by a Constant

For any constant k:

k · Σ aₙxⁿ = Σ (k · aₙ)xⁿ

with the same radius of convergence.

Differentiation

The derivative of a power series can be found by differentiating term by term:

d/dx[Σ aₙxⁿ] = Σ n·aₙx^(n-1)

The radius of convergence remains the same.

Integration

Power series can be integrated term by term:

∫[Σ aₙxⁿ]dx = C + Σ [aₙ/(n+1)]x^(n+1)

The radius of convergence remains the same.

Example 2: Operations on Power Series

For the power series f(x) = Σ(n=0 to ∞) xⁿ = 1 + x + x² + x³ + ... (which is valid for |x| < 1 and equals 1/(1-x)):

1. Derivative: f'(x) = Σ(n=1 to ∞) n·x^(n-1) = 1 + 2x + 3x² + ... = 1/(1-x)²

2. Integration: ∫f(x)dx = C + Σ(n=0 to ∞) x^(n+1)/(n+1) = C + x + x²/2 + x³/3 + ... = C - ln(1-x)

Both operations maintain the radius of convergence R = 1.

Taylor and Maclaurin Series

A Taylor series represents a function as a power series centered at a point x = a:

f(x) = Σ(n=0 to ∞) [f^(n)(a)/n!](x-a)ⁿ

where f^(n)(a) is the nth derivative of f evaluated at x = a.

A Maclaurin series is a special case of a Taylor series centered at a = 0:

f(x) = Σ(n=0 to ∞) [f^(n)(0)/n!]xⁿ

Common Maclaurin series expansions:

  • e^x = 1 + x + x²/2! + x³/3! + ... (for all x)
  • sin(x) = x - x³/3! + x⁵/5! - ... (for all x)
  • cos(x) = 1 - x²/2! + x⁴/4! - ... (for all x)
  • ln(1+x) = x - x²/2 + x³/3 - ... (for -1 < x ≤ 1)
  • 1/(1-x) = 1 + x + x² + x³ + ... (for |x| < 1)
Example 3: Finding a Taylor Series

Find the Taylor series for f(x) = ln(x) centered at a = 1.

We need to compute the derivatives of f(x) = ln(x) and evaluate them at x = 1:

f(x) = ln(x) → f(1) = ln(1) = 0

f'(x) = 1/x → f'(1) = 1

f''(x) = -1/x² → f''(1) = -1

f'''(x) = 2/x³ → f'''(1) = 2

f⁽⁴⁾(x) = -6/x⁴ → f⁽⁴⁾(1) = -6

In general, f^(n)(x) = (-1)^(n-1)(n-1)!/x^n and f^(n)(1) = (-1)^(n-1)(n-1)!

Therefore, the Taylor series is:

ln(x) = (x-1) - (x-1)²/2 + (x-1)³/3 - (x-1)⁴/4 + ... = Σ(n=1 to ∞) (-1)^(n-1)(x-1)^n/n

This series converges for 0 < x ≤ 2.

Applications of Power Series

Function Approximation

Taylor polynomials (finite sums of Taylor series terms) provide approximations for functions near the center point. The more terms used, the better the approximation.

Example 4: Function Approximation

Approximate e^0.2 using the first three terms of the Maclaurin series for e^x.

e^x ≈ 1 + x + x²/2 for x near 0

e^0.2 ≈ 1 + 0.2 + (0.2)²/2 = 1 + 0.2 + 0.02 = 1.22

The actual value is approximately 1.2214, so our approximation is quite good.

Integration of Difficult Functions

Power series can be used to integrate functions that do not have elementary antiderivatives.

Example 5: Integration Using Power Series

Evaluate ∫[0 to 0.5] (sin(x²)/x) dx.

Using the Maclaurin series for sin(x): sin(x) = x - x³/3! + x⁵/5! - ...

Substituting x² for x: sin(x²) = x² - x⁶/6 + x¹⁰/120 - ...

Therefore, sin(x²)/x = x - x⁵/6 + x⁹/120 - ...

Integrating term by term: ∫(sin(x²)/x) dx = x²/2 - x⁶/36 + x¹⁰/1200 - ...

Evaluating from 0 to 0.5: (0.5²/2 - 0.5⁶/36 + 0.5¹⁰/1200) - 0 ≈ 0.125 - 0.0002 + 0.0000 ≈ 0.1248

Solving Differential Equations

Power series methods can solve differential equations where standard techniques are difficult or impossible to apply.

Example 6: Solving a Differential Equation

Find a power series solution for y' = y with y(0) = 1.

Let y(x) = Σ(n=0 to ∞) aₙxⁿ = a₀ + a₁x + a₂x² + a₃x³ + ...

Then y'(x) = Σ(n=1 to ∞) n·aₙx^(n-1) = a₁ + 2a₂x + 3a₃x² + ...

The differential equation y' = y gives:

a₁ + 2a₂x + 3a₃x² + ... = a₀ + a₁x + a₂x² + a₃x³ + ...

Matching coefficients:

a₁ = a₀, 2a₂ = a₁, 3a₃ = a₂, ...

In general, (n+1)a(n+1) = aₙ, so a(n+1) = aₙ/(n+1)

With y(0) = 1, we have a₀ = 1

Therefore: a₁ = 1/1!, a₂ = 1/2!, a₃ = 1/3!, ...

The solution is y(x) = Σ(n=0 to ∞) xⁿ/n! = e^x
