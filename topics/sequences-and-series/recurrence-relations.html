Recurrence Relations

Understanding and solving equations that define sequences recursively

Home » Learning Resources » Mathematics Roadmap » Sequences and Series » Recurrence Relations

Introduction to Recurrence Relations

A recurrence relation is an equation that defines a sequence based on previous terms in the sequence, often with specified initial values. Instead of giving an explicit formula for the nth term, a recurrence relation describes how to calculate each term using previous terms.

Formally, a recurrence relation for a sequence {aₙ} has the form:

aₙ = f(a(n-1), a(n-2), ..., a(n-k))

where f is some function, and the sequence typically starts with given values for a₀, a₁, ..., a(k-1).

Recurrence relations appear in many contexts, including:

  • Mathematical sequences (Fibonacci, Lucas numbers)
  • Computer algorithms (divide-and-conquer, dynamic programming)
  • Economics (compound interest, population growth)
  • Physics (oscillatory systems, wave equations)
Example 1: The Fibonacci Sequence

The Fibonacci sequence is defined by the recurrence relation:

Fₙ = F(n-1) + F(n-2) for n ≥ 2, with F₀ = 0 and F₁ = 1

This generates the sequence: 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

Each term is the sum of the two preceding terms.

Types of Recurrence Relations

Linear Recurrence Relations

A linear recurrence relation of order k has the form:

aₙ = c₁a(n-1) + c₂a(n-2) + ... + cₖa(n-k) + g(n)

where c₁, c₂, ..., cₖ are constants (with cₖ ≠ 0) and g(n) is a function of n.

The relation is called:

  • Homogeneous if g(n) = 0 for all n
  • Non-homogeneous if g(n) ≠ 0 for some n
Example 2: Types of Linear Recurrence Relations

1. Homogeneous, first-order: aₙ = 2a(n-1)

2. Homogeneous, second-order: aₙ = 3a(n-1) - 2a(n-2)

3. Non-homogeneous, first-order: aₙ = 3a(n-1) + n²

4. Non-homogeneous, second-order: aₙ = a(n-1) + a(n-2) + 2ⁿ

Non-linear Recurrence Relations

Non-linear recurrence relations involve non-linear functions of previous terms. These are often more difficult to solve and may require numerical or approximation methods.

Example 3: Non-linear Recurrence Relations

1. aₙ = [a(n-1)]² (quadratic)

2. aₙ = a(n-1) · a(n-2) (product)

3. aₙ = 1/a(n-1) (reciprocal)

4. aₙ = √(a(n-1)) (square root)

Solving First-Order Linear Recurrence Relations

A first-order linear recurrence relation has the form:

aₙ = c · a(n-1) + g(n) for n ≥ 1

with initial value a₀.

Homogeneous Case (g(n) = 0)

When g(n) = 0, the relation becomes aₙ = c · a(n-1), which has the solution:

aₙ = c^n · a₀
Example 4: Homogeneous First-Order Relation

Solve the recurrence relation aₙ = 3a(n-1) with a₀ = 2.

This is a homogeneous first-order relation with c = 3.

Using the formula aₙ = c^n · a₀:

aₙ = 3^n · 2 = 2 · 3^n

The first few terms are: a₀ = 2, a₁ = 6, a₂ = 18, a₃ = 54, ...

Non-homogeneous Case

For non-homogeneous relations, several methods exist:

Method 1: Iteration

Expand the recurrence relation by substituting repeatedly, then identify a pattern.

Example 5: Solving by Iteration

Solve aₙ = 2a(n-1) + 3 with a₀ = 1.

Iterating:

a₁ = 2a₀ + 3 = 2(1) + 3 = 5

a₂ = 2a₁ + 3 = 2(5) + 3 = 13

a₃ = 2a₂ + 3 = 2(13) + 3 = 29

Continuing this pattern:

a₄ = 2a₃ + 3 = 2(29) + 3 = 61

Looking at the sequence 1, 5, 13, 29, 61, ..., we might notice that aₙ = 2^n + 2^n - 1 = 2^(n+1) - 1.

We can verify this formula satisfies our recurrence relation and initial condition.

Method 2: General Solution

The solution to aₙ = c · a(n-1) + g(n) with a₀ has the form:

aₙ = c^n · a₀ + Σ(k=1 to n) c^(n-k) · g(k)
Example 6: Using the General Formula

Solve aₙ = 2a(n-1) + 3 with a₀ = 1 using the general formula.

Here, c = 2 and g(k) = 3 for all k.

aₙ = 2^n · 1 + Σ(k=1 to n) 2^(n-k) · 3

= 2^n + 3 · Σ(k=1 to n) 2^(n-k)

= 2^n + 3 · 2^n · Σ(k=1 to n) 2^(-k)

= 2^n + 3 · 2^n · (1 - 2^(-n))/(1 - 2^(-1))

= 2^n + 3 · 2^n · (1 - 1/2^n)/(1/2)

= 2^n + 3 · 2^n · (2^n - 1)/2^n

= 2^n + 3(2^n - 1)

= 2^n + 3·2^n - 3

= 4·2^n - 3

= 2^(n+2) - 3

Solving Second-Order Linear Homogeneous Recurrence Relations

A second-order linear homogeneous recurrence relation has the form:

aₙ = c₁ · a(n-1) + c₂ · a(n-2) for n ≥ 2

with initial values a₀ and a₁.

Characteristic Equation Method

The solution involves finding the roots of the characteristic equation:

r² - c₁r - c₂ = 0

Case 1: Distinct roots r₁ and r₂

aₙ = α₁ · r₁^n + α₂ · r₂^n

where α₁ and α₂ are constants determined by the initial conditions.

Case 2: Repeated root r₁ = r₂

aₙ = α₁ · r₁^n + α₂ · n · r₁^n
Example 7: Distinct Roots

Solve aₙ = 5a(n-1) - 6a(n-2) with a₀ = 1 and a₁ = 4.

The characteristic equation is r² - 5r + 6 = 0, which factors as (r - 2)(r - 3) = 0.

So the roots are r₁ = 2 and r₂ = 3.

The general solution is aₙ = α₁ · 2^n + α₂ · 3^n.

Using the initial conditions:

a₀ = 1 = α₁ + α₂

a₁ = 4 = 2α₁ + 3α₂

Solving this system of equations: α₁ = -2, α₂ = 3

Therefore, aₙ = -2 · 2^n + 3 · 3^n

Example 8: Repeated Root

Solve aₙ = 6a(n-1) - 9a(n-2) with a₀ = 1 and a₁ = 6.

The characteristic equation is r² - 6r + 9 = 0, which factors as (r - 3)² = 0.

So the repeated root is r₁ = 3.

The general solution is aₙ = α₁ · 3^n + α₂ · n · 3^n.

Using the initial conditions:

a₀ = 1 = α₁

a₁ = 6 = 3α₁ + 3α₂

So α₁ = 1 and α₂ = 1

Therefore, aₙ = 3^n + n · 3^n = (n + 1) · 3^n

Generating Functions

A generating function is a powerful tool for solving recurrence relations by converting them into more manageable algebraic equations.

For a sequence {aₙ}, the ordinary generating function is defined as:

G(x) = a₀ + a₁x + a₂x² + a₃x³ + ... = Σ(n=0 to ∞) aₙx^n

Steps to solve recurrence relations using generating functions:

  1. Define the generating function G(x)
  2. Multiply the recurrence relation by x^n and sum over all valid n
  3. Express the resulting equation in terms of G(x)
  4. Solve for G(x)
  5. Expand G(x) as a power series to find the formula for aₙ
Example 9: Fibonacci Sequence using Generating Functions

Find the formula for the Fibonacci numbers using generating functions.

The Fibonacci recurrence relation is Fₙ = F(n-1) + F(n-2) for n ≥ 2, with F₀ = 0 and F₁ = 1.

Step 1: Define the generating function G(x) = Σ(n=0 to ∞) Fₙx^n

Step 2: Multiply the recurrence relation by x^n and sum:

Σ(n=2 to ∞) Fₙx^n = Σ(n=2 to ∞) F(n-1)x^n + Σ(n=2 to ∞) F(n-2)x^n

Step 3: Express in terms of G(x):

G(x) - F₀ - F₁x = x·Σ(n=2 to ∞) F(n-1)x^(n-1) + x²·Σ(n=2 to ∞) F(n-2)x^(n-2)

G(x) - 0 - x = x·(G(x) - F₀) + x²·G(x)

G(x) - x = x·G(x) + x²·G(x)

G(x) - x·G(x) - x²·G(x) = x

G(x)(1 - x - x²) = x

Step 4: Solve for G(x):

G(x) = x/(1 - x - x²)

Step 5: Using partial fractions:

G(x) = (1/√5)(1/(1 - φx) - 1/(1 - ψx)) where φ = (1 + √5)/2 and ψ = (1 - √5)/2

G(x) = (1/√5)Σ((φ^n - ψ^n)x^n)

Therefore, Fₙ = (φ^n - ψ^n)/√5

This is the Binet formula for Fibonacci numbers.

Applications of Recurrence Relations

Analysis of Algorithms

Recurrence relations are frequently used to analyze the time and space complexity of recursive algorithms.

Example 10: Binary Search

The time complexity T(n) of binary search on a sorted array of length n satisfies:

T(n) = T(n/2) + c

where c is a constant. With T(1) = c, the solution is T(n) = c · log₂n + c = O(log n).

Counting Problems

Many combinatorial problems can be expressed using recurrence relations.

Example 11: Tiling Problem

In how many ways can we tile a 2×n board using 1×2 and 2×2 tiles?

Let f(n) be the number of ways. We have:

f(n) = f(n-1) + f(n-2) for n ≥ 3, with f(1) = 1 and f(2) = 2

This is a Fibonacci-like recurrence with different initial values.

Financial Models

Recurrence relations model compound interest, loan amortization, and asset depreciation.

Example 12: Compound Interest

A bank account starts with $1000 and earns 5% annual interest. The balance after n years satisfies:

B(n) = 1.05 · B(n-1) for n ≥ 1, with B(0) = 1000

Solving: B(n) = 1000 · (1.05)^n

Population Dynamics

Recurrence relations model population growth, predator-prey interactions, and epidemics.

Example 13: Logistic Growth

A population with limited resources may follow the discrete logistic model:

P(n+1) = r · P(n) · (1 - P(n)/K)

where r is the growth rate and K is the carrying capacity.

This non-linear recurrence relation can exhibit complex behaviors, including chaos for certain values of r.
