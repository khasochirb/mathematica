Expectation & Variance

Measuring and analyzing the central tendency and dispersion of random variables

Home » Resources » Mathematics Roadmap » Probability » Expectation & Variance

Introduction to Expected Value

The expected value (or expectation or mean) of a random variable is a measure of the central tendency of its probability distribution. It represents the average value we would expect to observe if we could repeat the random experiment infinitely many times.

Definition for Discrete Random Variables

For a discrete random variable X with possible values x₁, x₂, ..., xₙ and corresponding probabilities p₁, p₂, ..., pₙ, the expected value is defined as:

E(X) = x₁p₁ + x₂p₂ + ... + xₙpₙ = Σ xᵢpᵢ

In other words, the expected value is the sum of each possible value multiplied by its probability.

Example: Expected Value of a Die Roll

Consider rolling a fair six-sided die. Let X be the number shown on the die. The possible values are 1, 2, 3, 4, 5, and 6, each with a probability of 1/6.

E(X) = 1×(1/6) + 2×(1/6) + 3×(1/6) + 4×(1/6) + 5×(1/6) + 6×(1/6)

E(X) = (1 + 2 + 3 + 4 + 5 + 6)/6 = 21/6 = 3.5

The expected value of a die roll is 3.5, even though it's impossible to actually roll a 3.5.

Definition for Continuous Random Variables

For a continuous random variable X with probability density function f(x), the expected value is defined as:

E(X) = ∫ x f(x) dx

where the integral is taken over the entire range of possible values of X.

Example: Expected Value of a Continuous Random Variable

Let X be a continuous random variable with probability density function f(x) = 2x for 0 ≤ x ≤ 1, and f(x) = 0 otherwise.

E(X) = ∫01 x (2x) dx = 2∫01 x² dx = 2[x³/3]01 = 2(1/3) = 2/3

The expected value of X is 2/3.

Properties of Expected Value

Linearity of Expectation

One of the most useful properties of expected value is its linearity. For any random variables X and Y, and any constants a and b:

E(aX + bY) = aE(X) + bE(Y)

This property holds regardless of whether X and Y are independent.

Example: Linearity of Expectation

If X represents the number shown when rolling a fair die, we know E(X) = 3.5.

Now, consider the random variable Y = 2X + 5, representing twice the die value plus 5.

Using linearity of expectation: E(Y) = E(2X + 5) = 2E(X) + 5 = 2(3.5) + 5 = 7 + 5 = 12

So the expected value of Y is 12.

Expected Value of a Function

For a function g(X) of a random variable X:

For discrete random variables: E(g(X)) = Σ g(xᵢ)p(xᵢ)
For continuous random variables: E(g(X)) = ∫ g(x)f(x) dx
Example: Expected Value of a Function

Let X be the result of rolling a fair die, and let g(X) = X².

E(X²) = 1²×(1/6) + 2²×(1/6) + 3²×(1/6) + 4²×(1/6) + 5²×(1/6) + 6²×(1/6)

E(X²) = (1 + 4 + 9 + 16 + 25 + 36)/6 = 91/6 ≈ 15.17

The expected value of X² is approximately 15.17.

Expected Value of a Product

For random variables X and Y:

E(XY) = E(X)E(Y) if X and Y are independent
E(XY) ≠ E(X)E(Y) if X and Y are dependent
Example: Expected Value of a Product

Let X and Y be the results of rolling two fair dice independently. We know E(X) = E(Y) = 3.5.

Since X and Y are independent: E(XY) = E(X)E(Y) = 3.5 × 3.5 = 12.25

Now, let Z be the sum of the dice values (Z = X + Y). If we define W = XZ, then X and Z are dependent (since Z includes X).

In this case, E(XZ) = E(X(X + Y)) = E(X² + XY) = E(X²) + E(XY) ≈ 15.17 + 12.25 = 27.42

This is different from E(X)E(Z) = 3.5 × (3.5 + 3.5) = 3.5 × 7 = 24.5

Variance and Standard Deviation

Definition of Variance

The variance of a random variable X, denoted Var(X) or σ², measures the spread or dispersion of the probability distribution around its expected value. It is defined as:

Var(X) = E[(X - E(X))²] = E(X²) - [E(X)]²

The second formula, Var(X) = E(X²) - [E(X)]², is often more convenient for calculations.

Definition of Standard Deviation

The standard deviation, denoted σ or SD(X), is the square root of the variance:

SD(X) = σ = √Var(X)

The standard deviation has the same units as the random variable itself, making it more interpretable than the variance for practical purposes.

Example: Calculating Variance and Standard Deviation

Continuing with our die-rolling example, we know:

E(X) = 3.5 and E(X²) ≈ 15.17

Using the formula Var(X) = E(X²) - [E(X)]²:

Var(X) = 15.17 - 3.5² = 15.17 - 12.25 = 2.92

SD(X) = √2.92 ≈ 1.71

The variance of a die roll is approximately 2.92, and the standard deviation is approximately 1.71.

Properties of Variance

For any random variables X and Y, and any constants a and b:

Var(aX + b) = a²Var(X)
Var(X + Y) = Var(X) + Var(Y) + 2Cov(X, Y)
If X and Y are independent: Var(X + Y) = Var(X) + Var(Y)

where Cov(X, Y) is the covariance between X and Y.

Example: Properties of Variance

For a random variable X representing a die roll, we found Var(X) ≈ 2.92.

For Y = 2X + 5:

Var(Y) = Var(2X + 5) = 2²Var(X) = 4 × 2.92 = 11.68

If X and Z are independent die rolls:

Var(X + Z) = Var(X) + Var(Z) = 2.92 + 2.92 = 5.84

Moments and Moment-Generating Functions

Moments of a Distribution

The kth moment of a random variable X about the origin is defined as E(X^k). The mean is the first moment (k = 1).

The kth central moment is defined as E[(X - μ)^k], where μ = E(X). The variance is the second central moment (k = 2).

Moment-Generating Function

The moment-generating function (MGF) of a random variable X is defined as:

MX(t) = E(e^{tX})

MGFs are useful because they uniquely determine a probability distribution and can be used to derive moments:

E(X^k) = d^k/dt^k [MX(t)] evaluated at t = 0
Example: Moment-Generating Function

For a standard normal random variable Z with μ = 0 and σ = 1, the MGF is:

MZ(t) = e^{t²/2}

To find E(Z²) (the second moment), we calculate:

d²/dt² [e^{t²/2}] = d/dt [te^{t²/2}] = e^{t²/2} + t×t×e^{t²/2} = e^{t²/2}(1 + t²)

Evaluating at t = 0 gives E(Z²) = 1, which confirms that the variance of a standard normal distribution is 1.

Example: Using Distribution Properties

In a binomial experiment with n = 10 trials and success probability p = 0.3:

E(X) = np = 10 × 0.3 = 3

Var(X) = np(1-p) = 10 × 0.3 × 0.7 = 2.1

SD(X) = √2.1 ≈ 1.45

On average, we expect 3 successes, with a standard deviation of approximately 1.45.

Applications of Expectation and Variance

Risk and Decision Making

Expected value plays a crucial role in decision theory and risk assessment. When faced with multiple choices, one approach is to select the option with the highest expected value. However, this doesn't account for risk aversion or preference.

The variance or standard deviation can be used as a measure of risk. Higher variance indicates greater uncertainty or volatility.

Example: Investment Decision

Consider two investment options:

Investment A: Expected return = 7%, Standard deviation = 5%

Investment B: Expected return = 10%, Standard deviation = 15%

While Investment B has a higher expected return, it also comes with significantly higher risk (higher standard deviation). A risk-averse investor might prefer Investment A despite its lower expected return.

Quality Control

In manufacturing, the mean and variance of product measurements are monitored to ensure quality. The mean should be close to the target specification, and the variance should be small to ensure consistency.

Example: Quality Control

A machine produces bolts with a target diameter of 10mm. Quality control measures the diameter of random samples and calculates:

Mean diameter = 10.02mm, Standard deviation = 0.05mm

If specifications require the diameter to be within 10 ± 0.15mm, we can calculate the probability of a bolt being out of specification using the normal distribution properties. With these parameters, approximately 99.7% of bolts would fall within 10.02 ± 3(0.05) = (9.87, 10.17), which meets the quality requirements.

Statistical Inference

The sample mean and variance are used to estimate the population mean and variance in statistical inference.

The Central Limit Theorem states that the distribution of sample means approaches a normal distribution as the sample size increases, regardless of the underlying population distribution.

Example: Sampling Distribution

A population has a mean μ = 50 and standard deviation σ = 10. If we take random samples of size n = 25:

The expected value of the sample mean is E(X̄) = μ = 50

The standard deviation of the sample mean (standard error) is σ/√n = 10/√25 = 10/5 = 2

By the Central Limit Theorem, X̄ is approximately normally distributed: X̄ ~ N(50, 4)

The probability that a sample mean falls between 46 and 54 is:

P(46 < X̄ < 54) = P(-2 < Z < 2) ≈ 0.9545

So about 95.45% of sample means will fall within 2 standard errors of the population mean.
