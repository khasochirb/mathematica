Linear & Quadratic Equations

Mastering fundamental equation types and their applications

Home » Learning Resources » Mathematics Roadmap » Algebra » Linear & Quadratic Equations

Linear Equations

A linear equation in one variable can be written in the standard form ax + b = 0, where a and b are constants, and a ≠ 0. Linear equations describe straight lines when graphed in a coordinate system.

Solving Linear Equations

The process typically involves:

  1. Moving all terms with the variable to one side of the equation
  2. Moving all constant terms to the other side
  3. Dividing both sides by the coefficient of the variable
Example: Solving a Linear Equation

Solve for x: 3x - 7 = 2x + 5

Step 1: Subtract 2x from both sides: 3x - 2x - 7 = 5

Step 2: Simplify: x - 7 = 5

Step 3: Add 7 to both sides: x = 12

Applications of Linear Equations

  • Simple Interest: I = Prt, where I is interest, P is principal, r is rate, and t is time
  • Distance, Speed, and Time: d = st, where d is distance, s is speed, and t is time
  • Linear Relationships: y = mx + b, where m is the slope and b is the y-intercept
  • Geometric Problems: Finding dimensions of shapes with given perimeter or area

Quadratic Equations

A quadratic equation can be written in the standard form ax2 + bx + c = 0, where a, b, and c are constants, and a ≠ 0. The graph of a quadratic equation is a parabola.

Methods for Solving Quadratic Equations

1. Factoring: If the quadratic expression can be factored, the equation can be solved by setting each factor equal to zero.

ax2 + bx + c = 0

If factored as (px + q)(rx + s) = 0, then px + q = 0 or rx + s = 0

2. Quadratic Formula: The solution to ax2 + bx + c = 0 is given by:

x = (-b ± √(b2 - 4ac)) / (2a)

3. Completing the Square: Transform the equation to the form (x + p)2 = q and then solve for x.

Example: Solving a Quadratic Equation

Solve: 2x2 - 7x + 3 = 0

Using the Quadratic Formula:

a = 2, b = -7, c = 3

x = (7 ± √(49 - 24)) / 4 = (7 ± √25) / 4 = (7 ± 5) / 4

x = 3 or x = 1/2

Alternative solution using factoring:

2x2 - 7x + 3 = 0

2x2 - 6x - x + 3 = 0

2x(x - 3) - 1(x - 3) = 0

(2x - 1)(x - 3) = 0

x = 1/2 or x = 3

The Discriminant

The discriminant, denoted by D = b2 - 4ac, determines the nature of the solutions to a quadratic equation ax2 + bx + c = 0:

  • If D > 0: Two distinct real solutions
  • If D = 0: One real solution (a repeated root)
  • If D < 0: Two complex conjugate solutions (no real solutions)
Example: Using the Discriminant

For the equation x2 - 4x + 5 = 0:

a = 1, b = -4, c = 5

D = (-4)2 - 4(1)(5) = 16 - 20 = -4

Since D < 0, the equation has no real solutions, only complex ones.

Graphical Interpretation

The graphical representations of linear and quadratic equations provide visual insights into their solutions and properties.

Linear Equations

  • Graph is a straight line
  • Slope = coefficient of x (m in y = mx + b)
  • y-intercept = constant term (b in y = mx + b)
  • Solution of ax + b = 0 is the x-intercept of the line

Quadratic Equations

  • Graph is a parabola
  • If a > 0, parabola opens upward; if a < 0, parabola opens downward
  • Vertex form: y = a(x - h)2 + k, where (h, k) is the vertex of the parabola
  • Solutions of ax2 + bx + c = 0 are the x-intercepts of the parabola
  • Axis of symmetry: x = -b/(2a)
Example: Analyzing a Parabola

For y = 2x2 - 8x + 6:

a = 2, b = -8, c = 6

Vertex x-coordinate: x = -b/(2a) = -(-8)/(2(2)) = 8/4 = 2

Vertex y-coordinate: y = 2(2)2 - 8(2) + 6 = 8 - 16 + 6 = -2

Vertex: (2, -2)

Since a > 0, the parabola opens upward

Applications of Quadratic Equations

Quadratic equations model many real-world situations:

Physics

  • Projectile Motion: The height h of an object thrown upward is given by h(t) = -16t2 + v0t + h0, where v0 is the initial velocity and h0 is the initial height
  • Free Fall: When an object falls from a height, its position is described by a quadratic equation

Economics

  • Revenue and Profit Functions: Often modeled as quadratic functions
  • Supply and Demand: Finding equilibrium price and quantity

Geometry

  • Area and Perimeter Problems: Finding dimensions of rectangles or other shapes given constraints
  • Pythagorean Theorem Applications: Many distance problems lead to quadratic equations
Example: Projectile Motion

A traditional Mongolian archer shoots an arrow upward from a height of 6 feet with an initial velocity of 40 feet per second. When will it hit the ground?

The height function is h(t) = -16t2 + 40t + 6

To find when it hits the ground, we set h(t) = 0:

-16t2 + 40t + 6 = 0

Using the quadratic formula with a = -16, b = 40, c = 6:

t = (-40 ± √(1600 - 384)) / (-32) = (-40 ± √1216) / (-32)

t = (-40 ± 34.87) / (-32)

t ≈ -0.16 or t ≈ 2.34

Since time cannot be negative in this context, the arrow hits the ground after approximately 2.34 seconds.
