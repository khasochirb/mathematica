Systems of Equations

Solving multiple equations simultaneously and their applications

Introduction to Systems of Equations

A system of equations consists of two or more equations with multiple variables that are to be solved simultaneously. The solution to a system is the set of values for the variables that satisfies all equations in the system.

Types of Systems of Equations

  • Linear Systems: All equations in the system are linear (e.g., 2x + 3y = 7 and 4x - y = 5)
  • Nonlinear Systems: At least one equation in the system is nonlinear (e.g., x² + y² = 25 and y = x + 5)

Number of Solutions

A system of equations can have:

  • Exactly one solution: The equations intersect at exactly one point
  • Infinitely many solutions: The equations represent the same relationship (dependent equations)
  • No solution: The equations are contradictory (inconsistent system)
Example: A Simple Linear System

Consider the system:

2x + y = 7

x - y = 1

This system has exactly one solution: (x, y) = (2, 3)

Solving Linear Systems

There are several methods for solving systems of linear equations:

1. Substitution Method

  1. Solve one equation for one variable in terms of the others
  2. Substitute this expression into the other equation(s)
  3. Solve the resulting equation
  4. Back-substitute to find the value of the other variable(s)
Example: Substitution Method

Solve the system:

2x + y = 7 ... (1)

x - y = 1 ... (2)

Step 1: Solve equation (2) for y: y = x - 1

Step 2: Substitute into equation (1): 2x + (x - 1) = 7

Step 3: Solve: 3x - 1 = 7, so 3x = 8, giving x = 8/3

Step 4: Back-substitute: y = 8/3 - 1 = 5/3

Solution: (x, y) = (8/3, 5/3)

2. Elimination Method

  1. Multiply one or both equations by constants to make the coefficients of one variable match in magnitude but opposite in sign
  2. Add the equations to eliminate that variable
  3. Solve the resulting equation for the remaining variable
  4. Substitute back to find the eliminated variable
Example: Elimination Method

Solve the system:

3x + 2y = 7 ... (1)

5x - 2y = 3 ... (2)

Step 1: Add equations (1) and (2): 8x = 10

Step 2: Solve for x: x = 5/4

Step 3: Substitute into equation (1): 3(5/4) + 2y = 7

Step 4: Solve for y: 15/4 + 2y = 7, so 2y = 7 - 15/4 = 28/4 - 15/4 = 13/4, giving y = 13/8

Solution: (x, y) = (5/4, 13/8)

3. Matrix Methods

For larger systems, matrix methods such as Gaussian elimination, Gauss-Jordan elimination, or using matrix inverses are often more efficient. These techniques are beyond the scope of this basic introduction.

Graphical Interpretation

Geometrically, a system of equations in two variables can be visualized as the intersection of curves in the Cartesian plane:

  • Exactly one solution: The curves intersect at exactly one point
  • Infinitely many solutions: The curves coincide (same equation)
  • No solution: The curves never intersect (e.g., parallel lines)

In Three Dimensions

For a system with three variables, each equation represents a plane in 3D space. The solution to the system is the intersection of these planes:

  • Three planes can intersect at a single point (unique solution)
  • Three planes can intersect along a line (infinitely many solutions)
  • Three planes can intersect at an entire plane (infinitely many solutions)
  • Three planes may have no common intersection point (no solution)

Solving Nonlinear Systems

When dealing with nonlinear systems, the techniques become more varied and sometimes more complex:

Common Methods

  • Substitution: Similar to linear systems, but may lead to higher-degree equations
  • Elimination: May require algebraic manipulation before variables can be eliminated
  • Graphical Method: Finding intersection points of curves
  • Numerical Methods: For complex systems that cannot be solved algebraically
Example: Nonlinear System

Solve the system:

x² + y² = 25 ... (1)

y = x + 5 ... (2)

Step 1: Substitute equation (2) into equation (1): x² + (x + 5)² = 25

Step 2: Expand: x² + x² + 10x + 25 = 25

Step 3: Simplify: 2x² + 10x = 0

Step 4: Factor: 2x(x + 5) = 0

Step 5: Solve: x = 0 or x = -5

Step 6: Substitute into equation (2):
If x = 0, then y = 0 + 5 = 5
If x = -5, then y = -5 + 5 = 0

Solutions: (0, 5) and (-5, 0)

Applications of Systems of Equations

Systems of equations model various real-world scenarios where multiple conditions or constraints exist simultaneously:

Mixture Problems

Problems involving mixtures of different substances or solutions at different concentrations.

Example: Mixture Problem

A chemist needs to mix a 20% acid solution with a 50% acid solution to obtain 12 liters of a 30% acid solution. How many liters of each solution should be used?

Let x = amount of 20% solution and y = amount of 50% solution

Volume equation: x + y = 12

Acid content equation: 0.20x + 0.50y = 0.30(12) = 3.6

Solving this system:
From the first equation: y = 12 - x
Substituting into the second equation: 0.20x + 0.50(12 - x) = 3.6
0.20x + 6 - 0.50x = 3.6
-0.30x + 6 = 3.6
-0.30x = -2.4
x = 8

Therefore, y = 12 - 8 = 4

The chemist should mix 8 liters of the 20% solution with 4 liters of the 50% solution.

Motion Problems

Problems involving speed, distance, and time relationships for moving objects.

Financial Applications

Problems involving investments with different interest rates, pricing with constraints, or cost-revenue analysis.

Economics

Supply and demand systems, market equilibrium, input-output models, and linear programming problems.

Engineering

Circuit analysis (Kirchhoff's laws), static equilibrium problems, and material balance equations.
