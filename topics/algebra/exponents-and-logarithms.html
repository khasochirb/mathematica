Exponents & Logarithms

Understanding the principles and applications of exponential and logarithmic functions

Home » Resources » Mathematics Roadmap » Algebra » Exponents & Logarithms

Exponents: Basic Principles

Exponents provide a concise way to express repeated multiplication. An expression like an means that the base a is multiplied by itself n times, where n is called the exponent or power.

Key Properties of Exponents

  • Product Rule: am × an = am+n
  • Quotient Rule: am ÷ an = am-n
  • Power Rule: (am)n = am×n
  • Power of a Product: (a×b)n = an × bn
  • Power of a Quotient: (a÷b)n = an ÷ bn
  • Zero Exponent: a0 = 1 (if a ≠ 0)
  • Negative Exponent: a-n = 1 ÷ an
Example: Simplifying Expressions with Exponents

Simplify: (32 × 34) ÷ 33

Using the product rule: 32 × 34 = 32+4 = 36

Using the quotient rule: 36 ÷ 33 = 36-3 = 33 = 27

Logarithms: Fundamentals

Logarithms are the inverse operations of exponents. If ax = b, then loga(b) = x. Here, a is the base of the logarithm, b is the argument, and x is the result.

Common Bases for Logarithms

  • Common Logarithm (base 10): Written as log(x) or log10(x)
  • Natural Logarithm (base e): Written as ln(x) or loge(x)
  • Binary Logarithm (base 2): Written as log2(x)

loga(b) = c means ac = b

Key Properties of Logarithms

  • Product Rule: loga(x × y) = loga(x) + loga(y)
  • Quotient Rule: loga(x ÷ y) = loga(x) - loga(y)
  • Power Rule: loga(xn) = n × loga(x)
  • Change of Base: loga(x) = logb(x) ÷ logb(a)
  • Logarithm of Base: loga(a) = 1
  • Logarithm of 1: loga(1) = 0
Example: Using Logarithm Properties

Simplify: log(2) + log(5)

Using the product rule: log(2) + log(5) = log(2 × 5) = log(10) = 1

Exponential & Logarithmic Functions

Exponential and logarithmic functions are essential in modeling various real-world phenomena like population growth, radioactive decay, and compound interest.

Exponential Functions

Exponential functions have the form f(x) = a × bx, where a and b are constants, and b > 0, b ≠ 1.

  • If b > 1, the function increases (grows) as x increases
  • If 0 < b < 1, the function decreases (decays) as x increases
  • The most important exponential function is ex, where e ≈ 2.71828 (Euler's number)

Logarithmic Functions

Logarithmic functions have the form f(x) = logb(x), where b > 0, b ≠ 1.

  • Domain: x > 0
  • Range: all real numbers
  • Logarithmic functions grow very slowly

The natural logarithm function ln(x) = loge(x) is the inverse of the exponential function ex.

ln(ex) = x and eln(x) = x

Solving Exponential & Logarithmic Equations

Solving equations involving exponents and logarithms often requires manipulation using their properties.

Exponential Equations

Common strategies:

  • Express both sides as powers of the same base
  • Take the logarithm of both sides
Example: Solving an Exponential Equation

Solve: 2x = 8

Since 8 = 23, we can rewrite: 2x = 23

Since the bases are the same, the exponents must be equal: x = 3

Alternative approach using logarithms:

Taking log base 10 of both sides: log(2x) = log(8)

Using the power rule: x × log(2) = log(8)

Solving for x: x = log(8) ÷ log(2) ≈ 3

Logarithmic Equations

Common strategies:

  • Combine logarithms using properties
  • Convert to exponential form
  • Check for extraneous solutions (values that make logarithm arguments negative or zero)
Example: Solving a Logarithmic Equation

Solve: log(x+3) + log(x-1) = 1

Using the product rule: log((x+3)(x-1)) = 1

Converting to exponential form: (x+3)(x-1) = 101 = 10

Expanding: x2 + 2x - 3 = 10

Rearranging: x2 + 2x - 13 = 0

Using the quadratic formula: x = (-2 ± √(4+52))/2 = (-2 ± √56)/2 = (-2 ± 2√14)/2 = -1 ± √14

Checking solutions: x = -1 + √14 ≈ 2.74 works, but x = -1 - √14 ≈ -4.74 gives negative arguments for logarithms

Therefore, the only solution is x = -1 + √14 ≈ 2.74

Applications of Exponents & Logarithms

Exponential and logarithmic functions have numerous real-world applications:

  • Compound Interest: P(t) = P0(1 + r/n)nt, where P0 is the principal, r is the interest rate, n is the number of times per year interest is compounded, and t is time in years
  • Continuous Compounding: P(t) = P0ert
  • Population Growth/Decay: P(t) = P0ekt, where k is the growth/decay rate constant
  • Radioactive Decay: N(t) = N0e-λt, where λ is the decay constant
  • Half-life: t1/2 = ln(2)/λ
  • pH in Chemistry: pH = -log[H+]
  • Richter Scale for Earthquakes: R = log(A/A0), where A is the amplitude of the earthquake and A0 is a standard reference value
  • Decibel Scale for Sound Intensity: dB = 10 × log(I/I0), where I is the intensity and I0 is a reference intensity
Example: Compound Interest

If you invest $5,000 at an annual interest rate of 4%, compounded quarterly, how much will you have after 10 years?

Using the compound interest formula: P(t) = P0(1 + r/n)nt

P(10) = 5000(1 + 0.04/4)4×10 = 5000(1.01)40 ≈ $7,436.43
