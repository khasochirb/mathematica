Regression and Correlation

Understanding relationships between variables

Introduction to Correlation

Correlation measures the strength and direction of the relationship between two variables. Unlike causation, correlation only identifies that two variables tend to move together, not that one causes the other.

Key characteristics of correlation:

  • Values range from -1 to +1
  • Positive correlation: variables move in the same direction
  • Negative correlation: variables move in opposite directions
  • Zero correlation: no linear relationship between variables
Example 1: Interpreting Correlation Coefficients

r = 0.92: Strong positive correlation (e.g., height and weight)

r = -0.85: Strong negative correlation (e.g., study time and error rate)

r = 0.15: Weak positive correlation (e.g., shoe size and intelligence)

r = 0: No linear correlation (e.g., eye color and typing speed)

Pearson Correlation Coefficient

The Pearson correlation coefficient (r) measures the linear relationship between two continuous variables. It is calculated as:

r = Σ[(xi - x̄)(yi - ȳ)] / √[Σ(xi - x̄)² · Σ(yi - ȳ)²]

Where:

  • xi and yi are individual data points
  • x̄ and ȳ are the means of x and y respectively

The coefficient of determination, r², indicates what proportion of the variance in y is explained by x.

Example 2: Calculating Pearson's r

Consider the data pairs: (2,8), (3,10), (5,12), (7,16), (8,20)

Step 1: Calculate the means: x̄ = 5, ȳ = 13.2

Step 2: Find deviations from means and their products:

  • (2-5)(8-13.2) = (-3)(-5.2) = 15.6
  • (3-5)(10-13.2) = (-2)(-3.2) = 6.4
  • (5-5)(12-13.2) = (0)(-1.2) = 0
  • (7-5)(16-13.2) = (2)(2.8) = 5.6
  • (8-5)(20-13.2) = (3)(6.8) = 20.4

Step 3: Sum the products: 15.6 + 6.4 + 0 + 5.6 + 20.4 = 48

Step 4: Calculate denominators:

Σ(xi - x̄)² = (-3)² + (-2)² + (0)² + (2)² + (3)² = 26

Σ(yi - ȳ)² = (-5.2)² + (-3.2)² + (-1.2)² + (2.8)² + (6.8)² = 107.2

Step 5: Calculate r = 48 / √(26 × 107.2) = 48 / 52.8 = 0.91

This indicates a strong positive correlation.

Spearman's Rank Correlation

Spearman's rank correlation assesses monotonic relationships between variables. It's useful when data is ordinal or the relationship is non-linear but still monotonic.

ρ = 1 - (6Σdi²) / [n(n² - 1)]

Where:

  • di is the difference between ranks of corresponding values
  • n is the number of data pairs
Example 3: Spearman's Rank Correlation

For the data: (100,85), (76,91), (82,89), (95,70), (68,65)

First, rank both variables (1 = highest):

X Y X Rank Y Rank d d²
100 85 1 2 -1 1
76 91 4 1 3 9
82 89 3 3 0 0
95 70 2 4 -2 4
68 65 5 5 0 0

Calculate ρ = 1 - (6 × 14) / [5(25 - 1)] = 1 - 84/120 = 0.3

This indicates a weak positive monotonic relationship.

Simple Linear Regression

Simple linear regression models the relationship between two variables using a straight line:

y = β₀ + β₁x + ε

Where:

  • y is the dependent variable (response)
  • x is the independent variable (predictor)
  • β₀ is the y-intercept
  • β₁ is the slope (change in y for a 1-unit change in x)
  • ε is the error term

Least Squares Method

The best-fitting line minimizes the sum of squared residuals (vertical distances between observed y values and predicted values). The parameters are calculated as:

β₁ = Σ[(xi - x̄)(yi - ȳ)] / Σ(xi - x̄)²
β₀ = ȳ - β₁x̄
Example 4: Simple Linear Regression

For the dataset: (2,5), (3,7), (5,8), (7,10), (8,12)

Calculate means: x̄ = 5, ȳ = 8.4

For the slope β₁:

Numerator: Σ[(xi - x̄)(yi - ȳ)] = (-3)(-3.4) + (-2)(-1.4) + (0)(-0.4) + (2)(1.6) + (3)(3.6) = 10.2 + 2.8 + 0 + 3.2 + 10.8 = 27

Denominator: Σ(xi - x̄)² = 9 + 4 + 0 + 4 + 9 = 26

β₁ = 27/26 = 1.04

For the intercept: β₀ = 8.4 - (1.04 × 5) = 3.2

Regression equation: y = 3.2 + 1.04x

Interpretation: For each 1-unit increase in x, y increases by 1.04 units on average.

Evaluating Regression Models

Residual Analysis

Residuals (ei = yi - ŷi) should be:

  • Normally distributed
  • Independent
  • Have constant variance (homoscedasticity)
  • Have mean zero

Coefficient of Determination (R²)

R² measures the proportion of variation in the dependent variable explained by the model:

R² = 1 - (SSE/SST) = SSR/SST

Where:

  • SSE = Σ(yi - ŷi)² (Sum of Squared Errors)
  • SST = Σ(yi - ȳ)² (Total Sum of Squares)
  • SSR = Σ(ŷi - ȳ)² (Regression Sum of Squares)

R² ranges from 0 to 1, with higher values indicating better fit.

Standard Error of the Estimate

Measures the average distance between observed values and the regression line:

Se = √[SSE/(n-2)]

Multiple Linear Regression

Multiple linear regression extends simple regression to include multiple predictors:

y = β₀ + β₁x₁ + β₂x₂ + ... + βₖxₖ + ε

Key considerations:

  • Each coefficient represents the effect of its predictor, holding all others constant
  • Multicollinearity: high correlation between predictors can cause unstable estimates
  • Adjusted R²: modified R² that penalizes for adding unnecessary predictors
Example 5: Multiple Regression Interpretation

A study models house prices based on size (sq ft) and age (years):

Price = 150,000 + 120(size) - 2,500(age)

Interpretation:

  • Base price: $150,000
  • Each additional square foot adds $120 to the price, holding age constant
  • Each additional year of age reduces the price by $2,500, holding size constant

Nonlinear Regression Models

When relationships between variables are not linear, alternative models include:

Polynomial Regression

y = β₀ + β₁x + β₂x² + β₃x³ + ... + ε

Exponential Model

y = β₀eβ₁x + ε

Logarithmic Model

y = β₀ + β₁ln(x) + ε

Power Model

y = β₀xβ₁ + ε
Example 6: Polynomial Regression

For data showing a curved relationship between temperature (x) and reaction rate (y), a quadratic model might be appropriate:

y = 2.3 + 1.5x - 0.08x²

This model captures how reaction rate initially increases with temperature but eventually decreases at high temperatures.

Logistic Regression

Logistic regression models the probability of a binary outcome as a function of predictor variables:

ln(p/(1-p)) = β₀ + β₁x₁ + β₂x₂ + ... + βₖxₖ

Where p is the probability of the outcome event occurring.

The predicted probability is:

p = 1 / (1 + e-(β₀ + β₁x₁ + β₂x₂ + ... + βₖxₖ))

Coefficients represent the change in log odds for a one-unit change in the predictor.

Example 7: Logistic Regression

A medical study models the probability of heart disease (yes/no) based on age and cholesterol level:

ln(p/(1-p)) = -9.2 + 0.08(age) + 0.02(cholesterol)

Interpretation of coefficients:

  • For each additional year of age, the log odds of heart disease increase by 0.08
  • For each additional unit of cholesterol, the log odds of heart disease increase by 0.02

In terms of odds ratios:

  • Age: e0.08 = 1.083, meaning the odds increase by 8.3% per year
  • Cholesterol: e0.02 = 1.020, meaning the odds increase by 2% per unit

Applications of Regression and Correlation

Prediction

Using regression equations to estimate the expected value of a dependent variable based on predictors.

Understanding Relationships

Quantifying the strength and direction of relationships between variables to gain insights into processes.

Process Optimization

Identifying critical factors that influence outcomes to improve processes and systems.

Risk Assessment

Identifying factors associated with increased risk of negative outcomes.

Example 8: Real-World Applications
Finance

Regression models predict stock returns based on market indicators and economic factors.

Healthcare

Logistic regression identifies risk factors for diseases, helping in early intervention.

Marketing

Multiple regression models customer behavior based on demographic and behavioral data.

Environmental Science

Regression models predict pollution levels based on industrial activity and weather conditions.
