Basic Operations with Complex Numbers

Understanding the fundamental operations and properties of complex numbers

Home » Resources » Mathematics Roadmap » Complex Numbers » Basic Operations

Introduction to Complex Numbers

Complex numbers extend the real number system by introducing the imaginary unit i, where i² = -1. A complex number is expressed in the form a + bi, where a and b are real numbers:

  • a is called the real part of the complex number
  • b is called the imaginary part of the complex number
  • i is the imaginary unit

Complex numbers arise naturally in mathematics when solving equations such as x² = -1, which has no solution in the real number system. With complex numbers, we expand our mathematical toolkit to solve problems that would otherwise be impossible.

The Complex Plane

Complex numbers can be represented geometrically on the complex plane (also called the Argand plane), where:

  • The horizontal axis represents the real part
  • The vertical axis represents the imaginary part

Each complex number z = a + bi corresponds to the point (a, b) on this plane.

Example: Visualizing Complex Numbers

The complex number z = 3 + 4i is represented as the point (3, 4) on the complex plane.

Similarly, z = -2 - 3i corresponds to the point (-2, -3).

Pure real numbers like z = 5 lie on the real axis at (5, 0).

Pure imaginary numbers like z = 2i lie on the imaginary axis at (0, 2).

Equality and Basic Properties

Two complex numbers are equal if and only if their real parts are equal and their imaginary parts are equal:

a + bi = c + di if and only if a = c and b = d

Complex Conjugate

The complex conjugate of a complex number z = a + bi, denoted by z̄ (z-bar), is defined as:

z̄ = a - bi

Properties of complex conjugates:

  • z + z̄ = 2a (twice the real part)
  • z - z̄ = 2bi (twice the imaginary part)
  • z · z̄ = a² + b² (the square of the absolute value of z)
  • (z̄) = z (the conjugate of the conjugate is the original number)
  • z + w = z̄ + w̄ (conjugate of a sum equals sum of conjugates)
  • z · w = z̄ · w̄ (conjugate of a product equals product of conjugates)
Example: Complex Conjugates

Let z = 3 - 2i. Find z̄ and compute z + z̄, z - z̄, and z · z̄.

The complex conjugate of z = 3 - 2i is z̄ = 3 + 2i.

z + z̄ = (3 - 2i) + (3 + 2i) = 6 (twice the real part)

z - z̄ = (3 - 2i) - (3 + 2i) = -4i (twice the imaginary part)

z · z̄ = (3 - 2i)(3 + 2i) = 9 + 6i - 6i - 4i² = 9 + 4 = 13 (the square of the absolute value of z)

Absolute Value (Modulus)

The absolute value or modulus of a complex number z = a + bi, denoted by |z|, is defined as:

|z| = √(a² + b²)

The absolute value represents the distance from the origin to the point z in the complex plane.

Example: Absolute Value

Find the absolute value of z = 3 + 4i.

|z| = √(3² + 4²) = √(9 + 16) = √25 = 5

This means the distance from the origin to the point (3, 4) on the complex plane is 5 units, which is consistent with the Pythagorean theorem.

Basic Arithmetic Operations

Addition and Subtraction

To add or subtract complex numbers, we add or subtract their real and imaginary parts separately:

(a + bi) + (c + di) = (a + c) + (b + d)i

(a + bi) - (c + di) = (a - c) + (b - d)i

Example: Addition and Subtraction

Let z₁ = 2 + 3i and z₂ = 4 - 5i.

z₁ + z₂ = (2 + 3i) + (4 - 5i) = (2 + 4) + (3 - 5)i = 6 - 2i

z₁ - z₂ = (2 + 3i) - (4 - 5i) = (2 - 4) + (3 - (-5))i = -2 + 8i

Multiplication

To multiply complex numbers, we use the distributive property and remember that i² = -1:

(a + bi)(c + di) = ac + adi + bci + bdi²

= ac + adi + bci - bd

= (ac - bd) + (ad + bc)i

Example: Multiplication

Multiply z₁ = 2 + 3i and z₂ = 4 - 5i.

(2 + 3i)(4 - 5i) = 2(4 - 5i) + 3i(4 - 5i)

= 8 - 10i + 12i - 15i²

= 8 - 10i + 12i - 15(-1)

= 8 - 10i + 12i + 15

= (8 + 15) + (-10 + 12)i

= 23 + 2i

Division

Division of complex numbers is accomplished by multiplying both the numerator and denominator by the complex conjugate of the denominator, which gives us a real number in the denominator:

(a + bi)/(c + di) = [(a + bi)(c - di)]/[(c + di)(c - di)]

= [(ac + bd) + (bc - ad)i]/(c² + d²)

Example: Division

Divide z₁ = 2 + 3i by z₂ = 4 - 5i.

(2 + 3i)/(4 - 5i) = [(2 + 3i)(4 + 5i)]/[(4 - 5i)(4 + 5i)]

= [(2)(4) + (2)(5i) + (3i)(4) + (3i)(5i)]/[16 + 25]

= [8 + 10i + 12i + 15i²]/41

= [8 + 22i + 15(-1)]/41

= [8 - 15 + 22i]/41

= [-7 + 22i]/41

= -7/41 + (22/41)i

≈ -0.171 + 0.537i

Powers of the Imaginary Unit

Since i² = -1, we can determine the pattern of powers of i:

i¹ = i

i² = -1

i³ = i² · i = -1 · i = -i

i⁴ = i³ · i = -i · i = -i² = -(-1) = 1

i⁵ = i⁴ · i = 1 · i = i

This pattern repeats with a cycle of 4, so for any integer n:

i^n has the same value as i^(n mod 4)

Example: Powers of i

Calculate i²³.

We need to find the remainder when 23 is divided by 4:

23 ÷ 4 = 5 with remainder 3

So i²³ = i³ = -i

Calculate i¹⁰⁰.

100 ÷ 4 = 25 with remainder 0

So i¹⁰⁰ = i⁰ = 1

Solving Equations with Complex Numbers

Complex numbers enable us to solve equations that have no real solutions. In particular, every polynomial equation of degree n has exactly n solutions in the complex number system (counting multiplicity). This is known as the Fundamental Theorem of Algebra.

Example: Solving Quadratic Equations

Solve the equation x² + 4 = 0.

Rearranging: x² = -4

Taking the square root of both sides: x = ±√(-4) = ±√4 · √(-1) = ±2i

Therefore, the solutions are x = 2i and x = -2i.

Example: Complex Roots of a Polynomial

Find all solutions to the equation x³ - 8 = 0.

Rearranging: x³ = 8

Taking the cube root: x = ∛8 = 2

However, a cubic equation must have three solutions. The other two solutions are complex:

x = 2(cos(120°) + i·sin(120°)) = 2(-1/2 + i·√3/2) = -1 + i·√3

x = 2(cos(240°) + i·sin(240°)) = 2(-1/2 - i·√3/2) = -1 - i·√3

Verifying: (-1 + i·√3)³ = -8, and (-1 - i·√3)³ = -8.
