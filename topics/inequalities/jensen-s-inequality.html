Jensen's Inequality

A powerful inequality for convex and concave functions

Home » Learning Resources » Mathematics Roadmap » Inequalities » Jensen's Inequality

Introduction to Jensen's Inequality

Jensen's inequality is a cornerstone result in convex analysis that establishes a relationship between the value of a convex function applied to an average of points and the average of the function applied to each point individually. Named after the Danish mathematician Johan Ludwig Jensen, who proved it in 1906, this inequality has become fundamental in various mathematical fields.

The inequality is particularly powerful because it unifies and generalizes many important inequalities in mathematics, including the arithmetic mean-geometric mean inequality, Hölder's inequality, and the Cauchy-Schwarz inequality. Its applications span across analysis, probability theory, information theory, and economics.

Convexity and Concavity

Before diving into Jensen's inequality, it's essential to understand convex and concave functions:

Convex Function: A function f is convex if, for any two points x and y in its domain and any t ∈ [0,1], the following holds:

f(tx + (1-t)y) ≤ tf(x) + (1-t)f(y)

Geometrically, this means that the line segment between any two points on the graph of the function lies above or on the graph. Examples include f(x) = x², e^x, and |x|.

Concave Function: A function f is concave if (-f) is convex, meaning:

f(tx + (1-t)y) ≥ tf(x) + (1-t)f(y)

Geometrically, the line segment between any two points on the graph lies below or on the graph. Examples include f(x) = log(x), √x, and -x².

Statement of Jensen's Inequality

Discrete Version

For a convex function f, if x₁, x₂, ..., xₙ are points in the domain of f, and λ₁, λ₂, ..., λₙ are non-negative weights that sum to 1, then:

f(λ₁x₁ + λ₂x₂ + ... + λₙxₙ) ≤ λ₁f(x₁) + λ₂f(x₂) + ... + λₙf(xₙ)

If f is strictly convex, equality holds if and only if x₁ = x₂ = ... = xₙ or if all but one of the λᵢ are zero.

For a concave function, the inequality is reversed:

f(λ₁x₁ + λ₂x₂ + ... + λₙxₙ) ≥ λ₁f(x₁) + λ₂f(x₂) + ... + λₙf(xₙ)

Continuous Version

Jensen's inequality can be extended to continuous distributions. If X is a random variable and f is a convex function, then:

f(E[X]) ≤ E[f(X)]

where E[X] denotes the expected value of X.

For a concave function, the inequality is reversed:

f(E[X]) ≥ E[f(X)]

Example: Verifying Jensen's Inequality for a Simple Case

Let's verify Jensen's inequality for the convex function f(x) = x² with points x₁ = 1 and x₂ = 3, using equal weights λ₁ = λ₂ = 0.5.

Left side: f(λ₁x₁ + λ₂x₂) = f(0.5 × 1 + 0.5 × 3) = f(2) = 2² = 4

Right side: λ₁f(x₁) + λ₂f(x₂) = 0.5 × 1² + 0.5 × 3² = 0.5 × 1 + 0.5 × 9 = 0.5 + 4.5 = 5

Indeed, 4 ≤ 5, confirming that f(λ₁x₁ + λ₂x₂) ≤ λ₁f(x₁) + λ₂f(x₂) for this convex function.

This makes intuitive sense: the average of the squared values (5) is greater than the square of the average value (4).

Proof of Jensen's Inequality

Geometric Proof for the Case n = 2

For two points x₁ and x₂ with weights λ₁ and λ₂ (where λ₁ + λ₂ = 1), Jensen's inequality states that:

f(λ₁x₁ + λ₂x₂) ≤ λ₁f(x₁) + λ₂f(x₂)

This can be proven geometrically. Let's denote x = λ₁x₁ + λ₂x₂. The point (x, f(x)) lies on the graph of f. The weighted average λ₁f(x₁) + λ₂f(x₂) corresponds to a point on the line segment connecting (x₁, f(x₁)) and (x₂, f(x₂)).

By the definition of convexity, this line segment lies above or on the graph of f. Therefore, f(x) ≤ λ₁f(x₁) + λ₂f(x₂).

Proof for the General Case

The proof for the general case can be done by induction or using properties of convex functions:

For a convex function f, there exists a supporting line at any point in its domain. For the point x₀ = λ₁x₁ + λ₂x₂ + ... + λₙxₙ, there exists a line l(x) = ax + b such that:

  1. l(x₀) = f(x₀)
  2. l(x) ≤ f(x) for all x in the domain of f

Using these properties:

λ₁f(x₁) + λ₂f(x₂) + ... + λₙf(xₙ) ≥ λ₁l(x₁) + λ₂l(x₂) + ... + λₙl(xₙ)

= l(λ₁x₁ + λ₂x₂ + ... + λₙxₙ) (since l is linear)

= l(x₀) = f(x₀)

This proves that f(λ₁x₁ + λ₂x₂ + ... + λₙxₙ) ≤ λ₁f(x₁) + λ₂f(x₂) + ... + λₙf(xₙ).

Proof for the Continuous Case

The proof for the continuous case follows from the discrete case by approximating the distribution of the random variable with a discrete distribution and taking the limit.

Example: Supporting Lines and Jensen's Inequality

Consider the convex function f(x) = e^x. At the point x₀ = 0, the supporting line is the tangent line l(x) = x + 1.

For points x₁ = -1 and x₂ = 1 with equal weights λ₁ = λ₂ = 0.5, we have x₀ = λ₁x₁ + λ₂x₂ = 0.

Jensen's inequality states: f(0) ≤ 0.5f(-1) + 0.5f(1)

i.e., e^0 ≤ 0.5e^(-1) + 0.5e^1

or 1 ≤ 0.5(e^(-1) + e^1) ≈ 0.5(0.368 + 2.718) ≈ 1.543

which is true, confirming Jensen's inequality.

Applications of Jensen's Inequality

Applications in Analysis and Inequalities

Jensen's inequality serves as a powerful tool for proving many classical inequalities:

1. AM-GM Inequality: Using Jensen's inequality with the concave function f(x) = log(x) gives:

log((x₁ + x₂ + ... + xₙ)/n) ≥ (log(x₁) + log(x₂) + ... + log(xₙ))/n

Taking the exponential of both sides yields the AM-GM inequality.

2. Hölder's Inequality: For 1/p + 1/q = 1 with p > 1, Hölder's inequality states:

∑(aᵢbᵢ) ≤ (∑aᵢᵖ)^(1/p) · (∑bᵢᵍ)^(1/q)

This can be derived using Jensen's inequality with appropriate convex functions.

3. Cauchy-Schwarz Inequality: A special case of Hölder's inequality when p = q = 2, which can also be derived from Jensen's inequality.

Applications in Probability and Statistics

1. Variance and Moments: For a random variable X, Jensen's inequality implies:

Var(X) = E[X²] - E[X]² ≥ 0

This follows from applying Jensen's inequality to the convex function f(x) = x².

2. Markov's and Chebyshev's Inequalities: These probability inequalities can be derived using Jensen's inequality.

3. Information Theory: In information theory, Jensen's inequality is used to prove that entropy is maximized for a uniform distribution and to establish properties of the Kullback-Leibler divergence.

Example: Entropy Maximization

For a discrete random variable X with n possible values and probability mass function p(x), the entropy is given by:

H(X) = -∑p(xᵢ)log(p(xᵢ))

Using Jensen's inequality with the concave function f(x) = -log(x), we can show that H(X) ≤ log(n), with equality when p(xᵢ) = 1/n for all i.

This proves that a uniform distribution maximizes entropy, a fundamental result in information theory.

Applications in Economics and Finance

1. Utility Theory: In economics, Jensen's inequality explains risk aversion in utility theory. If a utility function is concave, then:

U(E[X]) ≥ E[U(X)]

This means a risk-averse individual prefers a certain payoff E[X] to a random payoff X with the same expected value.

2. Option Pricing: Jensen's inequality is used in the context of risk-neutral pricing in finance, particularly for understanding the relationship between futures and forward prices under different interest rate models.

Extensions and Generalizations

Extensions of Jensen's Inequality

1. Jensen's Inequality for Multiple Functions: For convex functions f₁, f₂, ..., fₘ and a concave function g:

g(E[f₁(X)], E[f₂(X)], ..., E[fₘ(X)]) ≥ E[g(f₁(X), f₂(X), ..., fₘ(X))]

2. Conditional Jensen's Inequality: If X and Y are random variables and f is convex:

f(E[X|Y]) ≤ E[f(X)|Y]

3. Jensen's Inequality for Functions of Several Variables: The inequality extends to convex functions of multiple variables.

Related Concepts

1. Strict Convexity and Equality Conditions: If f is strictly convex, equality in Jensen's inequality holds if and only if X is almost surely constant.

2. Jensen's Inequality and Midpoint Convexity: A function satisfying f((x+y)/2) ≤ (f(x) + f(y))/2 for all x, y (midpoint convexity) is not necessarily convex without additional assumptions like continuity.

3. Jensen's Inequality in Functional Analysis: The inequality can be extended to infinite-dimensional spaces and operator theory.

Example: Generalized Jensen's Inequality in Information Theory

In information theory, the data processing inequality, which states that processing data cannot increase information content, can be derived from a generalized form of Jensen's inequality.

For random variables X → Y → Z forming a Markov chain, the mutual information satisfies:

I(X;Z) ≤ I(X;Y)

This shows that information is generally lost, not gained, when data is processed, a fundamental principle in communication theory and statistics.
