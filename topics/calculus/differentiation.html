Differentiation & Applications

Understanding derivatives and their real-world applications

Home » Learning Resources » Mathematics Roadmap » Calculus » Differentiation & Applications

The Derivative Concept

The derivative of a function represents its instantaneous rate of change. Geometrically, it gives the slope of the tangent line to the function's graph at a given point.

Formal Definition

The derivative of a function f at a point x = a is defined as:

f'(a) = limh→0 [f(a+h) - f(a)] / h

or equivalently:

f'(x) = limΔx→0 [f(x+Δx) - f(x)] / Δx

Notation

  • f'(x) - Lagrange's notation
  • df/dx - Leibniz's notation
  • y' - Newton's notation
  • Df(x) - Operator notation
Example: Finding a Derivative from the Definition

Find the derivative of f(x) = x² using the definition.

f'(x) = limh→0 [(x+h)² - x²] / h

= limh→0 [x² + 2xh + h² - x²] / h

= limh→0 [2xh + h²] / h

= limh→0 [2x + h]

= 2x

Therefore, f'(x) = 2x

Differentiation Rules

These rules allow us to differentiate complex functions without having to use the limit definition each time.

Basic Differentiation Rules

  • Constant Rule: If f(x) = c, then f'(x) = 0
  • Power Rule: If f(x) = xn, then f'(x) = n·xn-1
  • Constant Multiple Rule: If f(x) = c·g(x), then f'(x) = c·g'(x)
  • Sum/Difference Rule: If f(x) = g(x) ± h(x), then f'(x) = g'(x) ± h'(x)
  • Product Rule: If f(x) = g(x)·h(x), then f'(x) = g'(x)·h(x) + g(x)·h'(x)
  • Quotient Rule: If f(x) = g(x)/h(x), then f'(x) = [g'(x)·h(x) - g(x)·h'(x)]/[h(x)]²
  • Chain Rule: If f(x) = g(h(x)), then f'(x) = g'(h(x))·h'(x)

Derivatives of Common Functions

  • d/dx(sin x) = cos x
  • d/dx(cos x) = -sin x
  • d/dx(tan x) = sec² x
  • d/dx(ex) = ex
  • d/dx(ln x) = 1/x
  • d/dx(ax) = ax·ln a
  • d/dx(loga x) = 1/(x·ln a)
Example: Using Differentiation Rules

Find the derivative of f(x) = x³·sin x.

Using the product rule: f'(x) = (x³)'·sin x + x³·(sin x)'

= 3x²·sin x + x³·cos x

Example: Using the Chain Rule

Find the derivative of f(x) = sin(x²).

Let g(x) = sin x and h(x) = x²

Using the chain rule: f'(x) = g'(h(x))·h'(x)

= cos(x²)·2x

= 2x·cos(x²)

Higher Order Derivatives

The derivative of a derivative is called the second derivative, denoted by f''(x) or d²f/dx². This process can continue to create higher order derivatives.

Physical Interpretations

  • First derivative (f'): Rate of change (velocity in motion problems)
  • Second derivative (f''): Rate of change of the rate of change (acceleration)
  • Third derivative (f'''): Rate of change of acceleration (jerk)
Example: Finding Higher Derivatives

Find the first three derivatives of f(x) = x4 - 2x³ + 3x² - x + 5.

f'(x) = 4x³ - 6x² + 6x - 1

f''(x) = 12x² - 12x + 6

f'''(x) = 24x - 12

Implicit Differentiation

Sometimes a function is defined implicitly by an equation, rather than explicitly as y = f(x). Implicit differentiation allows us to find the derivative in such cases.

Process of Implicit Differentiation

  1. Differentiate each term of the equation with respect to x
  2. When differentiating terms involving y, use the chain rule, multiplying by dy/dx
  3. Solve for dy/dx
Example: Implicit Differentiation

Find dy/dx for the circle x² + y² = 25.

Differentiating both sides with respect to x:

d/dx(x² + y²) = d/dx(25)

2x + 2y·dy/dx = 0

2y·dy/dx = -2x

dy/dx = -x/y

Applications of Derivatives

Finding Extrema

Critical points occur when f'(x) = 0 or f'(x) is undefined. These points can be:

  • Local maximum: If f'(x) changes from positive to negative
  • Local minimum: If f'(x) changes from negative to positive
  • Inflection point: If f'(x) doesn't change sign

The Second Derivative Test: At a critical point where f'(x) = 0:

  • If f''(x) < 0, the point is a local maximum
  • If f''(x) > 0, the point is a local minimum
  • If f''(x) = 0, the test is inconclusive
Example: Finding Local Extrema

Find the local extrema of f(x) = x³ - 6x² + 9x + 1.

f'(x) = 3x² - 12x + 9

Setting f'(x) = 0:
3x² - 12x + 9 = 0
3(x² - 4x + 3) = 0
3(x - 1)(x - 3) = 0

So x = 1 or x = 3

f''(x) = 6x - 12

f''(1) = 6 - 12 = -6 < 0, so x = 1 gives a local maximum

f''(3) = 18 - 12 = 6 > 0, so x = 3 gives a local minimum

f(1) = 1 - 6 + 9 + 1 = 5

f(3) = 27 - 54 + 27 + 1 = 1

Therefore, there's a local maximum of 5 at x = 1 and a local minimum of 1 at x = 3.

Curve Sketching

Derivatives help identify key features of a graph:

  • Increasing/Decreasing: f is increasing when f'(x) > 0, decreasing when f'(x) < 0
  • Concavity: f is concave up when f''(x) > 0, concave down when f''(x) < 0
  • Inflection Points: Points where concavity changes, where f''(x) = 0 or f''(x) is undefined
Example: Curve Sketching

Sketch the curve f(x) = x³ - 3x² + 1.

f'(x) = 3x² - 6x = 3x(x - 2)

f'(x) = 0 when x = 0 or x = 2

f'(x) > 0 when x < 0 or x > 2 (f is increasing)

f'(x) < 0 when 0 < x < 2 (f is decreasing)

f''(x) = 6x - 6 = 6(x - 1)

f''(x) = 0 when x = 1

f''(x) < 0 when x < 1 (concave down)

f''(x) > 0 when x > 1 (concave up)

So there's an inflection point at x = 1

f(0) = 1, f(2) = -3, f(1) = -1

The curve has a local maximum at (0, 1), a local minimum at (2, -3), and an inflection point at (1, -1).

Optimization Problems

Many real-world problems involve finding maximum or minimum values of quantities.

Steps for Solving Optimization Problems

  1. Identify the quantity to be maximized or minimized
  2. Express this quantity as a function of one variable
  3. Find the domain of the function
  4. Find the critical points by setting the derivative equal to zero
  5. Determine which critical point gives the maximum or minimum value
  6. Verify the answer makes sense in the context of the problem
Example: Optimization Problem

A Mongolian herder wants to fence a rectangular area for their livestock along a straight river. If the side along the river needs no fence and the herder has 1000 meters of traditional wooden fencing, what are the dimensions that maximize the grazing area for their animals?

Let x be the width of the area (perpendicular to the river) and y be the length (parallel to the river).

Area: A = xy

Constraint: 2x + y = 1000 (since no fence is needed along the river)

Therefore, y = 1000 - 2x

Substituting: A(x) = x(1000 - 2x) = 1000x - 2x²

To maximize A, find where A'(x) = 0:

A'(x) = 1000 - 4x = 0

x = 250

Therefore, y = 1000 - 2(250) = 500

To verify this is a maximum, note that A''(x) = -4 < 0

The dimensions that maximize the grazing area are: width = 250 meters and length = 500 meters, giving an area of 125,000 square meters.

Related Rates

Related rates problems involve finding how the rate of change of one quantity relates to the rate of change of other quantities.

Steps for Solving Related Rates Problems

  1. Identify all variables and their rates of change
  2. Find an equation relating the variables
  3. Implicitly differentiate with respect to time
  4. Substitute known values and solve for the unknown rate
Example: Related Rates

A traditional Mongolian yurt is being constructed with a height of 3 meters and base radius of 4 meters. The walls are being raised at a rate of 0.5 meters per minute. How fast is the volume of the yurt increasing when the walls are 2 meters high?

Let V be the volume of the yurt, h be the height of the walls, and r be the radius of the base.

Given: dh/dt = 0.5 m/min, h = 2 m

We need to find dV/dt

The volume of a yurt can be approximated as a cylinder: V = πr²h

Differentiating with respect to time: dV/dt = πr²·dh/dt

When h = 2: dV/dt = π(4)²·0.5 = 8π ≈ 25.13 m³/min

Therefore, the volume of the yurt is increasing at approximately 25.13 cubic meters per minute when the walls are 2 meters high.
