Limits & Continuity

The foundational concepts of calculus

Introduction to Limits

Limits form the foundation of calculus. They allow us to describe the behavior of a function as its input approaches a particular value, even if the function is not defined at that value.

Informal Definition

The notation limx→a f(x) = L means that the value of f(x) gets arbitrarily close to L as x approaches a from either direction.

We write: limx→a f(x) = L

This means that for any ε > 0, there exists δ > 0 such that:

|f(x) - L| < ε whenever 0 < |x - a| < δ

One-sided Limits

  • Left-hand limit: limx→a- f(x) is the limit as x approaches a from values less than a
  • Right-hand limit: limx→a+ f(x) is the limit as x approaches a from values greater than a

For a limit to exist, both one-sided limits must exist and be equal:

limx→a f(x) = L if and only if limx→a- f(x) = limx→a+ f(x) = L

Example: Finding a Limit

Find limx→2 (x² - 4) / (x - 2)

Direct substitution leads to 0/0, which is an indeterminate form.

Using algebraic manipulation:

(x² - 4) / (x - 2) = ((x - 2)(x + 2)) / (x - 2) = x + 2 for x ≠ 2

As x approaches 2, x + 2 approaches 4

Therefore, limx→2 (x² - 4) / (x - 2) = 4

Properties of Limits

Limits follow several useful properties that make calculations more manageable:

Basic Limit Properties

If limx→a f(x) = L and limx→a g(x) = M, then:

  • Sum: limx→a [f(x) + g(x)] = L + M
  • Difference: limx→a [f(x) - g(x)] = L - M
  • Product: limx→a [f(x) × g(x)] = L × M
  • Quotient: limx→a [f(x) / g(x)] = L / M (if M ≠ 0)
  • Constant Multiple: limx→a [k × f(x)] = k × L (for any constant k)
  • Power: limx→a [f(x)]n = Ln (for any integer n, or for positive real n if L > 0)

Special Limits

  • limx→0 (sin x) / x = 1
  • limx→0 (1 - cos x) / x = 0
  • limx→0 (ex - 1) / x = 1
  • limx→0 (1 + x)1/x = e
  • limx→∞ (1 + 1/x)x = e
Example: Using Limit Properties

Find limx→3 (2x³ - 5x + 1)

Using the sum, difference, and product properties:

limx→3 (2x³ - 5x + 1) = 2 × limx→3 x³ - 5 × limx→3 x + limx→3 1

= 2 × 27 - 5 × 3 + 1

= 54 - 15 + 1 = 40

Techniques for Finding Limits

Direct Substitution

For most continuous functions, simply substitute the value:

limx→a f(x) = f(a)

Factoring

When encountering indeterminate forms like 0/0, try factoring and canceling common terms.

Rationalization

Multiply numerator and denominator by a conjugate to eliminate radicals in indeterminate forms.

L'Hôpital's Rule

For indeterminate forms of type 0/0 or ∞/∞, the limit equals the limit of the ratio of derivatives:

limx→a f(x)/g(x) = limx→a f'(x)/g'(x)

Example: Using Rationalization

Find limx→4 (√x - 2) / (x - 4)

Direct substitution gives 0/0, an indeterminate form.

Rationalize the numerator:

(√x - 2) / (x - 4) × (√x + 2) / (√x + 2) = (x - 4) / ((x - 4)(√x + 2)) = 1 / (√x + 2) for x ≠ 4

Now we can substitute: limx→4 1 / (√x + 2) = 1 / (√4 + 2) = 1 / 4 = 0.25

Example: Using L'Hôpital's Rule

Find limx→0 (ex - 1 - x) / x²

Direct substitution gives 0/0.

Applying L'Hôpital's rule:

limx→0 (ex - 1 - x) / x² = limx→0 (ex - 1) / 2x

This is still 0/0, so apply L'Hôpital's rule again:

limx→0 (ex) / 2 = 1/2

Limits at Infinity

Limits at infinity describe the behavior of functions as the input grows arbitrarily large (or negatively large).

limx→∞ f(x) = L means f(x) approaches L as x becomes arbitrarily large

limx→-∞ f(x) = L means f(x) approaches L as x becomes arbitrarily large in the negative direction

Important Limits at Infinity

  • For any rational function P(x)/Q(x) where the degree of P is less than the degree of Q: limx→±∞ P(x)/Q(x) = 0
  • For any polynomial, the limit at infinity is determined by the highest-degree term
  • limx→∞ xne-x = 0 for any positive n
  • limx→∞ (1 + k/x)x = ek for any constant k
Example: Limit at Infinity

Find limx→∞ (3x² + 2x - 5) / (x² + 7)

Divide both numerator and denominator by the highest power of x (which is x²):

limx→∞ (3 + 2/x - 5/x²) / (1 + 7/x²)

As x → ∞, all terms with x in the denominator approach 0:

limx→∞ (3 + 0 - 0) / (1 + 0) = 3

Continuity of Functions

Continuity is a fundamental property of functions that allows us to connect the concept of limits with the actual values of functions.

Definition of Continuity

A function f is continuous at a point a if all three conditions are met:

  1. f(a) is defined (the function exists at point a)
  2. limx→a f(x) exists (the limit exists as x approaches a)
  3. limx→a f(x) = f(a) (the limit equals the function value at a)

Types of Discontinuities

  • Removable Discontinuity: The limit exists but either f(a) is undefined or f(a) ≠ limx→a f(x)
  • Jump Discontinuity: The left and right limits exist but are not equal
  • Infinite Discontinuity: The limit is infinite (the function "blows up")
  • Oscillatory Discontinuity: The limit doesn't exist due to oscillation (e.g., sin(1/x) as x → 0)
Example: Analyzing Continuity

Analyze the continuity of f(x) = (x² - 9) / (x - 3) at x = 3.

We note that f(3) is undefined because direct substitution gives 0/0.

However, we can factor the numerator: (x² - 9) / (x - 3) = ((x - 3)(x + 3)) / (x - 3) = x + 3 for x ≠ 3

So limx→3 f(x) = limx→3 (x + 3) = 6

This is a removable discontinuity because the limit exists but f(3) is undefined.

We can make the function continuous by redefining it as:

g(x) = { x + 3 if x ≠ 3 6 if x = 3 }

The Intermediate Value Theorem

The Intermediate Value Theorem (IVT) is a key result that follows from continuity.

Intermediate Value Theorem: If f is continuous on a closed interval [a, b] and k is any value between f(a) and f(b), then there exists at least one value c in [a, b] such that f(c) = k.

This theorem has several important applications:

  • Finding Roots: If f is continuous and changes sign on an interval, it must have a root in that interval
  • Existence of Solutions: Guaranteeing that equations have solutions within certain intervals
  • Proof of Theoretical Results: Establishing the existence of various mathematical entities
Example: Applying the Intermediate Value Theorem

Show that the equation x³ - 2x - 5 = 0 has a solution in the interval [2, 3].

Let f(x) = x³ - 2x - 5

f(2) = 2³ - 2(2) - 5 = 8 - 4 - 5 = -1

f(3) = 3³ - 2(3) - 5 = 27 - 6 - 5 = 16

Since f(2) < 0 and f(3) > 0, and f is continuous (as a polynomial), by the Intermediate Value Theorem, there must be a value c in [2, 3] such that f(c) = 0.

Therefore, the equation has a solution in the interval [2, 3].
