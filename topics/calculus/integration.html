Antiderivatives and Indefinite Integrals

An antiderivative of a function f(x) is a function F(x) such that F'(x) = f(x). The indefinite integral, denoted by ∫f(x)dx, represents the family of all antiderivatives of f(x).

If F'(x) = f(x), then ∫f(x)dx = F(x) + C

where C is an arbitrary constant called the constant of integration.

Basic Integration Rules

  • ∫k·f(x)dx = k·∫f(x)dx (where k is a constant)
  • ∫[f(x) ± g(x)]dx = ∫f(x)dx ± ∫g(x)dx
  • ∫xndx = xn+1/(n+1) + C (n ≠ -1)
  • ∫1/x dx = ln|x| + C
  • ∫exdx = ex + C
  • ∫axdx = ax/ln(a) + C
  • ∫sin(x)dx = -cos(x) + C
  • ∫cos(x)dx = sin(x) + C
  • ∫tan(x)dx = ln|sec(x)| + C
  • ∫sec2(x)dx = tan(x) + C
Example: Finding Indefinite Integrals

Find ∫(3x² - 2x + 5)dx.

Using the linearity of integration:

∫(3x² - 2x + 5)dx = 3∫x²dx - 2∫xdx + 5∫dx

= 3(x³/3) - 2(x²/2) + 5x + C

= x³ - x² + 5x + C

Integration Techniques

1. Substitution Method (u-substitution)

This technique is based on the chain rule for differentiation and is used when an integral involves a composition of functions.

If u = g(x), then ∫f(g(x))g'(x)dx = ∫f(u)du

Example: Integration by Substitution

Find ∫x·cos(x²)dx.

Let u = x², so du = 2x dx, or x dx = du/2

∫x·cos(x²)dx = ∫cos(u)·(du/2) = (1/2)∫cos(u)du = (1/2)sin(u) + C = (1/2)sin(x²) + C

2. Integration by Parts

Based on the product rule for differentiation, integration by parts is used for integrals of the form ∫u(x)v'(x)dx.

∫u(x)v'(x)dx = u(x)v(x) - ∫v(x)u'(x)dx

or using the notation: ∫u dv = uv - ∫v du

Example: Integration by Parts

Find ∫x·ln(x)dx.

Let u = ln(x) and dv = x dx

Then du = (1/x)dx and v = x²/2

∫x·ln(x)dx = ln(x)·(x²/2) - ∫(x²/2)·(1/x)dx

= (x²/2)·ln(x) - (1/2)∫x dx

= (x²/2)·ln(x) - (x²/4) + C

= (x²/2)·ln(x) - (x²/4) + C

3. Partial Fractions

This technique is used for integrating rational functions (ratio of polynomials) by decomposing them into simpler fractions.

Example: Partial Fractions

Find ∫(3x - 2)/((x - 1)(x + 2))dx.

Express as partial fractions: (3x - 2)/((x - 1)(x + 2)) = A/(x - 1) + B/(x + 2)

Multiply all terms by (x - 1)(x + 2): 3x - 2 = A(x + 2) + B(x - 1)

When x = 1: 3(1) - 2 = A(1 + 2), so 1 = 3A, thus A = 1/3

When x = -2: 3(-2) - 2 = B(-2 - 1), so -8 = -3B, thus B = 8/3

Therefore: (3x - 2)/((x - 1)(x + 2)) = (1/3)/(x - 1) + (8/3)/(x + 2)

∫(3x - 2)/((x - 1)(x + 2))dx = (1/3)∫1/(x - 1)dx + (8/3)∫1/(x + 2)dx

= (1/3)ln|x - 1| + (8/3)ln|x + 2| + C

4. Trigonometric Substitution

This method is useful for integrals involving expressions like √(a² - x²), √(a² + x²), or √(x² - a²).

Example: Trigonometric Substitution

Find ∫1/√(4 - x²)dx.

Let x = 2sin(θ), so dx = 2cos(θ)dθ and √(4 - x²) = √(4 - 4sin²(θ)) = √(4cos²(θ)) = 2cos(θ)

∫1/√(4 - x²)dx = ∫1/(2cos(θ))·2cos(θ)dθ = ∫dθ = θ + C = arcsin(x/2) + C

Definite Integrals

A definite integral represents the signed area between the curve y = f(x) and the x-axis over an interval [a, b].

∫abf(x)dx = F(b) - F(a)

where F is an antiderivative of f.

Properties of Definite Integrals

  • ∫abf(x)dx = -∫baf(x)dx
  • ∫aaf(x)dx = 0
  • ∫abk·f(x)dx = k·∫abf(x)dx (where k is a constant)
  • ∫ab[f(x) ± g(x)]dx = ∫abf(x)dx ± ∫abg(x)dx
  • ∫abf(x)dx = ∫acf(x)dx + ∫cbf(x)dx
Example: Evaluating a Definite Integral

Evaluate ∫02(x² + 3x)dx.

First, find the antiderivative: ∫(x² + 3x)dx = (x³/3) + (3x²/2) + C

Apply the fundamental theorem of calculus:

∫02(x² + 3x)dx = [(2³/3) + (3·2²/2)] - [(0³/3) + (3·0²/2)]

= [8/3 + 6] - [0 + 0]

= 8/3 + 6 = 8/3 + 18/3 = 26/3

Applications of Integration

1. Area Between Curves

The area between two curves y = f(x) and y = g(x) from x = a to x = b is given by:

A = ∫ab[f(x) - g(x)]dx

where f(x) ≥ g(x) for all x in [a, b]

Example: Area Between Curves

Find the area of a traditional Mongolian pattern bounded by the curves y = x² and y = x + 2 from x = -1 to x = 2. This pattern is commonly used in traditional Mongolian decorative arts.

First, find where the curves intersect: x² = x + 2

x² - x - 2 = 0

(x - 2)(x + 1) = 0

x = 2 or x = -1

So the curves intersect at the endpoints of our interval.

We need to determine which function is greater in the interval. At x = 0:

f(0) = 0² = 0, g(0) = 0 + 2 = 2

Since g(0) > f(0), g(x) > f(x) throughout the interval.

A = ∫-12[(x + 2) - x²]dx = ∫-12(x + 2 - x²)dx

= [x²/2 + 2x - x³/3]-12

= [(2²/2 + 2·2 - 2³/3) - ((-1)²/2 + 2·(-1) - (-1)³/3)]

= [(2 + 4 - 8/3) - (1/2 - 2 + 1/3)]

= [6 - 8/3 - (-7/6)]

= 6 - 8/3 + 7/6 = 36/6 - 16/6 + 7/6 = 27/6 = 9/2

Therefore, the area of the traditional Mongolian pattern is 9/2 square units.

2. Volume of Revolution

The volume of the solid formed by revolving the region between y = f(x) and the x-axis from x = a to x = b around the x-axis is:

V = π·∫ab[f(x)]²dx

The volume of the solid formed by revolving the region around the y-axis is:

V = 2π·∫abx·f(x)dx

Example: Volume of Revolution

Find the volume of a traditional Mongolian bowl formed by revolving the region bounded by y = √x, y = 0, and x = 4 around the x-axis. This shape is commonly used in traditional Mongolian pottery.

Using the disk method:

V = π·∫04[√x]²dx = π·∫04x dx = π·[x²/2]04 = π·(8 - 0) = 8π cubic units

Therefore, the volume of the traditional Mongolian bowl is 8π cubic units.

3. Arc Length

The arc length of a curve y = f(x) from x = a to x = b is given by:

L = ∫ab√(1 + [f'(x)]²)dx

Example: Arc Length

Find the arc length of the curve y = x³/3 from x = 0 to x = 1.

f'(x) = x², so 1 + [f'(x)]² = 1 + x⁴

L = ∫01√(1 + x⁴)dx

This integral doesn't have an elementary antiderivative, so we would typically use numerical methods to evaluate it.

4. Work

Work done by a variable force F(x) moving an object from x = a to x = b is:

W = ∫abF(x)dx

Example: Work

A force of F(x) = 20 + 3x² newtons is applied to an object moving along a straight line from x = 1 to x = 4 meters. Find the work done.

W = ∫14(20 + 3x²)dx = ∫1420dx + ∫143x²dx

= 20[x]14 + 3[x³/3]14 = 20(4 - 1) + 3(64/3 - 1/3)

= 60 + 3(63/3) = 60 + 63 = 123 joules
