Introduction to Differential Equations

A differential equation is an equation that relates a function with its derivatives. In general, a differential equation models how a quantity changes with respect to one or more variables.

Classification of Differential Equations

  • Ordinary Differential Equations (ODEs): Contain derivatives with respect to a single independent variable
  • Partial Differential Equations (PDEs): Contain derivatives with respect to multiple independent variables

Order and Degree

  • Order: The highest derivative that appears in the equation
  • Degree: The power to which the highest derivative is raised (when the equation is polynomial in all derivatives)

Linear vs. Nonlinear

A differential equation is linear if it can be written in the form:

an(x)y(n) + an-1(x)y(n-1) + ... + a1(x)y' + a0(x)y = f(x)

where ai(x) and f(x) are functions of the independent variable x only.

Any equation that cannot be written in this form is nonlinear.

Example: Classifying Differential Equations

1. dy/dx = 2x - 3y

Order: 1 (first derivative only)

Type: Linear (can be written as dy/dx + 3y = 2x)

2. d²y/dx² + (dy/dx)² = sin(x)

Order: 2 (second derivative appears)

Type: Nonlinear (due to the squared first derivative term)

First-Order Differential Equations

Separable Equations

A first-order differential equation is separable if it can be written in the form:

g(y)·dy/dx = f(x)

The solution method involves separating the variables and integrating both sides:

∫g(y)dy = ∫f(x)dx + C

Example: First-Order Differential Equation

In traditional Mongolian airag fermentation, the rate of lactic acid production is proportional to the current amount of lactic acid bacteria present. If we let y(t) represent the amount of lactic acid bacteria at time t (in hours), and k is the growth rate constant, then:

dy/dt = ky

This is a separable equation. Separating variables:

dy/y = k dt

Integrating both sides:

ln|y| = kt + C

Therefore, y = Cekt

If we know that y(0) = y₀ (initial amount of bacteria), then C = y₀, and the solution is:

y = y₀ekt

This exponential growth model describes how the lactic acid bacteria population grows during the traditional airag fermentation process.

Linear First-Order Equations

A linear first-order differential equation has the form:

dy/dx + P(x)y = Q(x)

The solution method involves finding an integrating factor μ(x) = e∫P(x)dx and then solving:

y·μ(x) = ∫Q(x)·μ(x)dx + C

Example: Solving a Linear First-Order Equation

Solve dy/dx + 2y = ex

This is in the form dy/dx + P(x)y = Q(x) with P(x) = 2 and Q(x) = ex

The integrating factor is μ(x) = e∫2dx = e2x

Multiplying both sides by e2x: e2x·dy/dx + 2y·e2x = e3x

The left side is d/dx(y·e2x), so: d/dx(y·e2x) = e3x

Integrating both sides: y·e2x = ∫e3xdx = e3x/3 + C

Solving for y: y = e-2x·(e3x/3 + C) = ex/3 + Ce-2x

Exact Equations

An equation of the form M(x,y)dx + N(x,y)dy = 0 is exact if there exists a function F(x,y) such that:

∂F/∂x = M(x,y) and ∂F/∂y = N(x,y)

The condition for exactness is: ∂M/∂y = ∂N/∂x

The solution is F(x,y) = C, where C is a constant.

Example: Solving an Exact Equation

Solve (2xy + y²)dx + (x² + 2xy)dy = 0

Let's check if this is exact:

M(x,y) = 2xy + y² and N(x,y) = x² + 2xy

∂M/∂y = 2x + 2y and ∂N/∂x = 2x + 2y

Since ∂M/∂y = ∂N/∂x, the equation is exact.

To find F(x,y), we integrate ∂F/∂x = M(x,y) with respect to x:

F(x,y) = ∫(2xy + y²)dx = x²y + xy² + h(y)

where h(y) is a function of y only.

Now, ∂F/∂y = x² + 2xy + h'(y) = N(x,y) = x² + 2xy

Therefore, h'(y) = 0, so h(y) = C₁ (a constant)

The solution is F(x,y) = x²y + xy² + C₁ = C

or equivalently: x²y + xy² = C (where C is a new constant)

Second-Order Differential Equations

Homogeneous Linear Equations with Constant Coefficients

A second-order homogeneous linear equation with constant coefficients has the form:

ay'' + by' + cy = 0

where a, b, and c are constants (a ≠ 0).

The solution method involves finding the roots of the characteristic equation:

ar² + br + c = 0

The nature of the roots determines the form of the solution:

  • Two distinct real roots r₁ and r₂: y = C₁er₁x + C₂er₂x
  • One repeated real root r: y = C₁erx + C₂xerx
  • Complex conjugate roots α ± βi: y = eαx(C₁cos(βx) + C₂sin(βx))
Example: Solving a Second-Order Homogeneous Equation

Solve y'' - 5y' + 6y = 0

The characteristic equation is r² - 5r + 6 = 0

Using the quadratic formula: r = (5 ± √(25 - 24))/2 = (5 ± 1)/2

So r₁ = 3 and r₂ = 2

The general solution is y = C₁e3x + C₂e2x

Non-homogeneous Linear Equations

A non-homogeneous linear equation has the form:

ay'' + by' + cy = g(x)

The general solution is the sum of:

  • The general solution of the corresponding homogeneous equation (complementary function)
  • A particular solution of the non-homogeneous equation

Methods for finding a particular solution include:

  • Method of Undetermined Coefficients: Applicable when g(x) is a polynomial, exponential, sine, cosine, or a combination of these
  • Variation of Parameters: A more general method applicable to a wider range of g(x)
Example: Solving a Non-homogeneous Equation

Solve y'' - 5y' + 6y = 3e2x

The complementary function (from the previous example) is yc = C₁e3x + C₂e2x

For the particular solution, since 2 is a root of the characteristic equation, we try yp = Axe2x

Substituting into the original equation:

yp = Axe2x

yp' = A(e2x + 2xe2x)

yp'' = A(2e2x + 2e2x + 4xe2x) = A(4e2x + 4xe2x)

Substituting into the original equation:

A(4e2x + 4xe2x) - 5A(e2x + 2xe2x) + 6Axe2x = 3e2x

Simplifying: A(4e2x + 4xe2x - 5e2x - 10xe2x + 6xe2x) = 3e2x

A(4e2x - 5e2x + 4xe2x - 10xe2x + 6xe2x) = 3e2x

A(-e2x) = 3e2x

A = -3

Therefore, yp = -3xe2x

The general solution is y = yc + yp = C₁e3x + C₂e2x - 3xe2x

Applications of Differential Equations

Growth and Decay Models

Many natural phenomena can be modeled by differential equations:

  • Exponential Growth: dP/dt = kP (for population, compound interest)
  • Exponential Decay: dN/dt = -kN (for radioactive decay, drug elimination)
  • Logistic Growth: dP/dt = kP(1 - P/M) (for population with limited resources)
Example: Radioactive Decay

Radioactive carbon-14 has a half-life of 5,730 years. If an artifact initially contained 100 grams of carbon-14, how much remains after 10,000 years?

The differential equation for decay is dN/dt = -kN

This has solution N(t) = N₀e-kt

To find k, we use the half-life: N₀/2 = N₀e-k·5730

1/2 = e-5730k

ln(1/2) = -5730k

k = ln(2)/5730 ≈ 0.000121

Now, N(10000) = 100e-0.000121·10000 ≈ 100e-1.21 ≈ 100 · 0.298 ≈ 29.8 grams

Mechanical Systems

Differential equations model various mechanical systems:

  • Simple Harmonic Motion: m(d²x/dt²) + kx = 0
  • Damped Harmonic Motion: m(d²x/dt²) + c(dx/dt) + kx = 0
  • Forced Harmonic Motion: m(d²x/dt²) + c(dx/dt) + kx = F₀cos(ωt)
Example: Spring-Mass System

A 2-kg mass is attached to a spring with spring constant k = 8 N/m. If the mass is pulled 0.5 meters from equilibrium and released, describe its motion.

The differential equation is: 2(d²x/dt²) + 8x = 0

or d²x/dt² + 4x = 0

This has characteristic equation r² + 4 = 0, with roots r = ±2i

The general solution is x(t) = C₁cos(2t) + C₂sin(2t)

Given that x(0) = 0.5 and x'(0) = 0 (released from rest):

x(0) = C₁ = 0.5

x'(t) = -2C₁sin(2t) + 2C₂cos(2t)

x'(0) = 2C₂ = 0, so C₂ = 0

Therefore, x(t) = 0.5cos(2t), which represents simple harmonic motion with amplitude 0.5 meters and angular frequency 2 rad/s (or period π seconds).

Electrical Circuits

Differential equations model electrical circuits:

  • RC Circuit: dq/dt + (1/RC)q = E(t)/R
  • RLC Circuit: L(d²q/dt²) + R(dq/dt) + (1/C)q = E(t)
Example: RC Circuit

In an RC circuit with R = 1000 Ω and C = 10-6 F, a battery with voltage E = 12 V is connected at t = 0. Find the charge q(t) on the capacitor.

The differential equation is: dq/dt + (1/RC)q = E/R

dq/dt + 1000q = 12·10³

This is a first-order linear equation with P(t) = 1000 and Q(t) = 12·10³

The integrating factor is μ(t) = e∫1000dt = e1000t

d/dt(q·e1000t) = 12·10³·e1000t

q·e1000t = 12·10³·e1000t/1000 + C = 12·e1000t + C

q(t) = 12 + Ce-1000t

Given that q(0) = 0: 0 = 12 + C, so C = -12

Therefore, q(t) = 12(1 - e-1000t) μC

This shows the capacitor charges exponentially toward its maximum charge of 12 μC.
