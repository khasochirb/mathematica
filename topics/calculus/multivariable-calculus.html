Multivariable Calculus

Partial Derivatives, Gradient, Divergence, and Multiple Integrals

Functions of Multiple Variables

A function of multiple variables assigns a single value to an ordered collection of inputs. For example, a function f(x,y) maps each point (x,y) in its domain to a value f(x,y).

Graphical Representation

For a function f(x,y) of two variables, the graph is a surface in three-dimensional space. Each point (x,y,z) on the surface satisfies z = f(x,y).

Level Curves and Level Surfaces

  • Level Curves: For a function f(x,y), a level curve is the set of all points (x,y) where f(x,y) = k for some constant k. Level curves are the 2D analog of contour lines on a topographic map.
  • Level Surfaces: For a function f(x,y,z), a level surface is the set of all points (x,y,z) where f(x,y,z) = k for some constant k.
Example: Level Curves

Find the level curves of f(x,y) = x² + y².

The level curves are given by x² + y² = k for constants k ≥ 0.

These are circles centered at the origin with radius √k.

For example:
k = 1: x² + y² = 1 (unit circle)
k = 4: x² + y² = 4 (circle of radius 2)
k = 9: x² + y² = 9 (circle of radius 3)

Partial Derivatives

Partial derivatives measure how a function changes as one of its variables changes while others remain constant.

Definition

For a function f(x,y), the partial derivatives are:

∂f/∂x = limh→0 [f(x+h,y) - f(x,y)]/h

∂f/∂y = limh→0 [f(x,y+h) - f(x,y)]/h

Notation

Several notations are used for partial derivatives:

  • ∂f/∂x, fx, Dxf
  • ∂f/∂y, fy, Dyf

Computation

To compute a partial derivative, treat all other variables as constants and use the rules of single-variable calculus.

Example: Partial Derivatives

Consider a traditional Mongolian ger with a cylindrical base of radius r and height h, topped by a conical roof of height H. The total surface area A is given by:

A(r,h,H) = 2πrh + πr√(r² + H²)

Find the partial derivatives with respect to r, h, and H:

∂A/∂r = 2πh + π√(r² + H²) + πr²/√(r² + H²)

∂A/∂h = 2πr

∂A/∂H = πrH/√(r² + H²)

These partial derivatives tell us how the surface area changes with respect to each dimension of the ger. For example, ∂A/∂h = 2πr shows that increasing the height of the cylindrical base by one unit increases the surface area by the circumference of the base.

Higher-Order Partial Derivatives

We can take partial derivatives of partial derivatives:

  • ∂²f/∂x² = ∂/∂x(∂f/∂x) (second derivative with respect to x)
  • ∂²f/∂y² = ∂/∂y(∂f/∂y) (second derivative with respect to y)
  • ∂²f/∂x∂y = ∂/∂x(∂f/∂y) (mixed partial derivative)
  • ∂²f/∂y∂x = ∂/∂y(∂f/∂x) (mixed partial derivative)

For most "well-behaved" functions, the mixed partial derivatives are equal: ∂²f/∂x∂y = ∂²f/∂y∂x. This is known as Clairaut's theorem.

Example: Higher-Order Partial Derivatives

For f(x,y) = x³y² + xy - y³, find ∂²f/∂x∂y.

We already found ∂f/∂y = 2x³y + x - 3y²

Now, ∂²f/∂x∂y = ∂/∂x(∂f/∂y) = ∂/∂x(2x³y + x - 3y²) = 6x²y + 1

The Gradient and Directional Derivatives

The Gradient Vector

The gradient of a scalar function f(x,y,z) is a vector field that points in the direction of steepest increase of the function:

∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z)

Properties of the gradient:

  • The gradient is perpendicular to level curves (in 2D) or level surfaces (in 3D)
  • The magnitude of the gradient gives the rate of change in the direction of steepest ascent
Example: Finding the Gradient

Find the gradient of f(x,y,z) = xy² + yz + z²x.

∂f/∂x = y² + z²

∂f/∂y = 2xy + z

∂f/∂z = y + 2zx

Therefore, ∇f = (y² + z², 2xy + z, y + 2zx)

Directional Derivatives

The directional derivative represents the rate of change of a function in a given direction.

Duf = ∇f · u

where u is a unit vector indicating the direction.

Example: Directional Derivative

For f(x,y) = x² - 3xy + 2y², find the directional derivative at the point (1,2) in the direction of v = (3,4).

First, find the gradient:

∇f = (2x - 3y, -3x + 4y)

At the point (1,2): ∇f(1,2) = (2(1) - 3(2), -3(1) + 4(2)) = (-4, 5)

Normalize v to get a unit vector: |v| = √(3² + 4²) = 5

u = v/|v| = (3/5, 4/5)

The directional derivative is:

Duf(1,2) = ∇f(1,2) · u = (-4, 5) · (3/5, 4/5) = (-4)(3/5) + 5(4/5) = -12/5 + 20/5 = 8/5

Multiple Integrals

Double Integrals

A double integral ∫∫R f(x,y) dA represents the volume under the surface z = f(x,y) above the region R in the xy-plane.

Evaluation of double integrals:

∫∫R f(x,y) dA = ∫ab ∫g₁(x)g₂(x) f(x,y) dy dx

or equivalently,

∫∫R f(x,y) dA = ∫cd ∫h₁(y)h₂(y) f(x,y) dx dy

Example: Evaluating a Double Integral

Evaluate ∫∫R xy dA where R is the rectangle [0,2] × [1,3].

∫∫R xy dA = ∫02 ∫13 xy dy dx

= ∫02 x[y²/2]13 dx

= ∫02 x[(3²/2) - (1²/2)] dx

= ∫02 x[9/2 - 1/2] dx

= ∫02 4x dx

= 4[x²/2]02

= 4(2²/2 - 0) = 4(2) = 8

Triple Integrals

A triple integral ∫∫∫E f(x,y,z) dV represents the hypervolume of a four-dimensional object over a region E in 3D space.

Example: Triple Integral

Evaluate ∫∫∫E xyz dV where E is the cube [0,1] × [0,1] × [0,1].

∫∫∫E xyz dV = ∫01 ∫01 ∫01 xyz dz dy dx

= ∫01 ∫01 xy[z²/2]01 dy dx

= ∫01 ∫01 xy(1/2) dy dx

= (1/2) ∫01 x[y²/2]01 dx

= (1/2) ∫01 x(1/2) dx

= (1/4) ∫01 x dx

= (1/4)[x²/2]01

= (1/4)(1/2) = 1/8

Vector Calculus

Vector Fields

A vector field F assigns a vector to each point in a region of space. In 3D, F(x,y,z) = (F₁(x,y,z), F₂(x,y,z), F₃(x,y,z)).

Divergence

The divergence of a vector field F measures the rate at which "flow" is expanding or contracting at each point:

div F = ∇ · F = ∂F₁/∂x + ∂F₂/∂y + ∂F₃/∂z

Example: Calculating Divergence

Find the divergence of F(x,y,z) = (x²y, yz², xz).

∂F₁/∂x = ∂(x²y)/∂x = 2xy

∂F₂/∂y = ∂(yz²)/∂y = z²

∂F₃/∂z = ∂(xz)/∂z = x

div F = 2xy + z² + x

Curl

The curl of a vector field F measures the rotation or circulation at each point:

curl F = ∇ × F = (∂F₃/∂y - ∂F₂/∂z, ∂F₁/∂z - ∂F₃/∂x, ∂F₂/∂x - ∂F₁/∂y)

Example: Calculating Curl

Find the curl of F(x,y,z) = (x²y, yz², xz).

∂F₃/∂y - ∂F₂/∂z = ∂(xz)/∂y - ∂(yz²)/∂z = 0 - 2yz = -2yz

∂F₁/∂z - ∂F₃/∂x = ∂(x²y)/∂z - ∂(xz)/∂x = 0 - z = -z

∂F₂/∂x - ∂F₁/∂y = ∂(yz²)/∂x - ∂(x²y)/∂y = 0 - x² = -x²

curl F = (-2yz, -z, -x²)

Line Integrals

A line integral ∫C F · dr calculates the work done by a force field F along a curve C.

Green's Theorem

Green's theorem relates a line integral around a simple closed curve C to a double integral over the region D enclosed by C:

∮C (F₁ dx + F₂ dy) = ∫∫D (∂F₂/∂x - ∂F₁/∂y) dA

Stokes' Theorem

Stokes' theorem is a generalization of Green's theorem to 3D. It relates a line integral around a closed curve C to a surface integral over any surface S bounded by C:

∮C F · dr = ∫∫S (∇ × F) · n dS

Divergence Theorem

The divergence theorem relates a surface integral over a closed surface S to a triple integral over the region V enclosed by S:

∫∫S F · n dS = ∫∫∫V (∇ · F) dV
