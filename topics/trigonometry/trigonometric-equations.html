Trigonometric Equations

Solving equations involving trigonometric functions

Home » Learning Resources » Mathematics Roadmap » Trigonometry » Trigonometric Equations

Introduction to Trigonometric Equations

Trigonometric equations contain trigonometric functions and require us to find all values of the variable that satisfy the equation. These equations typically have infinitely many solutions due to the periodic nature of trigonometric functions.

The key to solving trigonometric equations is to:

  1. Simplify the equation using algebraic and trigonometric identities
  2. Isolate the trigonometric function
  3. Find the principal solutions (those in the interval [0, 2π) or [0°, 360°))
  4. Use the periodic nature of trigonometric functions to find the general solution

The general solutions typically include terms with 2πn (or 360°n) for integer values of n, reflecting the periodicity of the functions:

  • sin, csc: period = 2π (or 360°)
  • cos, sec: period = 2π (or 360°)
  • tan, cot: period = π (or 180°)
Example 1: Basic Trigonometric Equation

Solve the equation: sin(x) = 1/2

Step 1: Find the principal solutions in [0, 2π).

sin(x) = 1/2 means x = π/6 or x = 5π/6 (30° or 150°)

Step 2: Find the general solution by adding multiples of the period (2π).

x = π/6 + 2πn or x = 5π/6 + 2πn, where n is any integer

In degrees: x = 30° + 360°n or x = 150° + 360°n

Linear Trigonometric Equations

Linear trigonometric equations involve a single trigonometric function, often in the form a·trig(x) + b = 0, where a and b are constants.

Basic Form: trig(x) = c

The solutions depend on the value of c and the specific trigonometric function:

sin(x) = c (when -1 ≤ c ≤ 1)
• x = sin-1(c) + 2πn
• x = π - sin-1(c) + 2πn

cos(x) = c (when -1 ≤ c ≤ 1)
• x = cos-1(c) + 2πn
• x = -cos-1(c) + 2πn

tan(x) = c (any real c)
• x = tan-1(c) + πn
Example 2: Linear Trigonometric Equation

Solve for all solutions of 2cos(x) + 1 = 0 in the interval [0, 2π).

Step 1: Isolate the trigonometric function.

2cos(x) = -1

cos(x) = -1/2

Step 2: Find all values where cos(x) = -1/2 in [0, 2π).

The reference angle is cos-1(1/2) = π/3 (or 60°)

Since cos(x) is negative, x is in the second or third quadrant.

x = π - π/3 = 2π/3 ≈ 2.09 (or 120°)

x = π + π/3 = 4π/3 ≈ 4.19 (or 240°)

Step 3: Express the general solution.

x = 2π/3 + 2πn or x = 4π/3 + 2πn, where n is any integer

Equations Using Fundamental Identities

Many trigonometric equations can be solved by applying fundamental identities to simplify the expression.

Common Approaches:

  • Pythagorean Identities: sin²(x) + cos²(x) = 1, 1 + tan²(x) = sec²(x), 1 + cot²(x) = csc²(x)
  • Double-Angle Formulas: sin(2x) = 2sin(x)cos(x), cos(2x) = cos²(x) - sin²(x)
  • Half-Angle Formulas: sin²(x/2) = (1 - cos(x))/2, cos²(x/2) = (1 + cos(x))/2
  • Factoring: sin(x) - sin(y) = 2sin((x-y)/2)cos((x+y)/2)
Example 3: Using Identities

Solve the equation: sin²(x) - cos²(x) = 1

Step 1: Apply an appropriate identity.

Using sin²(x) + cos²(x) = 1, we can rewrite sin²(x) = 1 - cos²(x)

Therefore: (1 - cos²(x)) - cos²(x) = 1

1 - 2cos²(x) = 1

-2cos²(x) = 0

cos²(x) = 0

Step 2: Solve the simplified equation.

cos(x) = 0 occurs at x = π/2 and x = 3π/2 in [0, 2π)

Step 3: Express the general solution.

x = π/2 + πn, where n is any integer

In degrees: x = 90° + 180°n

Quadratic Trigonometric Equations

Quadratic trigonometric equations involve a squared trigonometric function or two different functions, often in the form a·trig²(x) + b·trig(x) + c = 0.

Solution Approaches:

  1. Factoring: Rewrite as a product of two expressions
  2. Quadratic Formula: Use the standard quadratic formula
  3. Substitution: Set u = trig(x) to convert to a standard quadratic equation
  4. Pythagorean Identities: Replace one function with another using identities
Example 4: Quadratic in Sine

Solve: 2sin²(x) - 3sin(x) + 1 = 0

Step 1: Use substitution u = sin(x) to convert to a standard quadratic.

2u² - 3u + 1 = 0

Step 2: Solve using the quadratic formula or factoring.

Factoring: (2u - 1)(u - 1) = 0

u = 1/2 or u = 1

Step 3: Substitute back to get sin(x).

sin(x) = 1/2 or sin(x) = 1

Step 4: Solve each simple equation.

For sin(x) = 1/2: x = π/6 + 2πn or x = 5π/6 + 2πn

For sin(x) = 1: x = π/2 + 2πn

Step 5: Combine all solutions.

x = π/6 + 2πn or x = 5π/6 + 2πn or x = π/2 + 2πn

Multiple-Angle Equations

Multiple-angle equations involve functions like sin(nx), cos(nx), or tan(nx) where n is an integer greater than 1.

Solution Approaches:

  1. Directly solve: Find principal solutions for the multiple angle
  2. Expand: Use multiple-angle formulas to rewrite in terms of the original angle
  3. Substitution: Let u = nx to simplify, then convert back to x
Example 5: Double-Angle Equation

Solve: sin(2x) = cos(x) for x in [0, 2π)

Step 1: Use a multiple-angle formula.

sin(2x) = 2sin(x)cos(x)

Step 2: Set up the equation.

2sin(x)cos(x) = cos(x)

cos(x)(2sin(x) - 1) = 0

Step 3: Solve each factor.

cos(x) = 0 or 2sin(x) - 1 = 0

cos(x) = 0 ⟹ x = π/2 or x = 3π/2

sin(x) = 1/2 ⟹ x = π/6 or x = 5π/6

Step 4: Check each solution in the original equation.

For x = π/6: sin(2π/6) = sin(π/3) = √3/2, cos(π/6) = √3/2 ✓

For x = π/2: sin(2π/2) = sin(π) = 0, cos(π/2) = 0 ✓

For x = 5π/6: sin(2×5π/6) = sin(5π/3) = -√3/2, cos(5π/6) = -√3/2 ✓

For x = 3π/2: sin(2×3π/2) = sin(3π) = 0, cos(3π/2) = 0 ✓

Step 5: List all solutions in [0, 2π).

x = π/6, π/2, 5π/6, 3π/2

Trigonometric Equations with Multiple Functions

These equations involve different trigonometric functions, often requiring the use of identities to convert everything to a single function.

Common Approaches:

  1. Express in terms of one function: Use quotient identities like tan(x) = sin(x)/cos(x)
  2. Pythagorean identities: Replace functions with their equivalents
  3. Angle addition formulas: If the equation involves functions of different angles
Example 6: Mixed Functions

Solve: sin(x) + cos(x) = 1 for x in [0, 2π)

Step 1: Square both sides to use a Pythagorean identity.

(sin(x) + cos(x))² = 1²

sin²(x) + 2sin(x)cos(x) + cos²(x) = 1

Step 2: Apply the Pythagorean identity sin²(x) + cos²(x) = 1.

1 + 2sin(x)cos(x) = 1

2sin(x)cos(x) = 0

Step 3: Solve the simplified equation.

sin(x) = 0 or cos(x) = 0

sin(x) = 0 ⟹ x = 0, π, 2π

cos(x) = 0 ⟹ x = π/2, 3π/2

Step 4: Check each solution in the original equation.

For x = 0: sin(0) + cos(0) = 0 + 1 = 1 ✓

For x = π/2: sin(π/2) + cos(π/2) = 1 + 0 = 1 ✓

For x = π: sin(π) + cos(π) = 0 + (-1) = -1 ✗

For x = 3π/2: sin(3π/2) + cos(3π/2) = -1 + 0 = -1 ✗

For x = 2π: sin(2π) + cos(2π) = 0 + 1 = 1 ✓

Step 5: List valid solutions in [0, 2π).

x = 0, π/2, 2π

Note: Since 2π = 0 in terms of angle measurement, the solutions are x = 0, π/2

Alternative approach: Divide by √2 and use substitution u = sin(x) + cos(x), v = sin(x) - cos(x)

Applications of Trigonometric Equations

Trigonometric equations have applications in many fields including physics, engineering, music, and medicine.

Key Applications:

  • Wave Motion: Sound waves, light waves, water waves
  • Oscillatory Motion: Pendulums, springs, AC circuits
  • Periodic Phenomena: Tides, planetary motion, seasonal variations
  • Signal Processing: Fourier analysis, filters, communications
Example 7: Applied Problem

A weight hanging from a spring oscillates up and down. Its height h (in cm) above its equilibrium position at time t (in seconds) is given by h = 5cos(2t). At what time(s) during the first 2 seconds will the weight be 2.5 cm above the equilibrium position?

Step 1: Set up the equation.

5cos(2t) = 2.5

cos(2t) = 1/2

Step 2: Solve the equation for 2t.

2t = ±π/3 + 2πn

Step 3: Solve for t with positive values (time cannot be negative).

t = π/6 + πn or t = -π/6 + πn (where n is an integer)

For t in [0, 2], we get: t = π/6 ≈ 0.52 seconds, t = 7π/6 ≈ 3.67 seconds

Since 3.67 > 2, only t ≈ 0.52 seconds is in our time range

Answer: The weight will be 2.5 cm above the equilibrium position at approximately 0.52 seconds during the first 2 seconds.
