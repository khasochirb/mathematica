Trigonometric Identities & Formulas

The essential relationships between trigonometric functions

Basic Trigonometric Functions

Trigonometric functions are defined in terms of the coordinates of a point on the unit circle or as ratios of the sides of a right triangle. For an angle θ in standard position:

sin(θ) = opposite / hypotenuse
cos(θ) = adjacent / hypotenuse
tan(θ) = opposite / adjacent = sin(θ) / cos(θ)
csc(θ) = 1 / sin(θ)
sec(θ) = 1 / cos(θ)
cot(θ) = 1 / tan(θ) = cos(θ) / sin(θ)

On the unit circle, for a point (x, y) corresponding to angle θ:

sin(θ) = y
cos(θ) = x
tan(θ) = y / x (when x ≠ 0)

The domains and ranges of these functions are important to understand:

  • sin(θ): Domain: all real numbers; Range: [-1, 1]
  • cos(θ): Domain: all real numbers; Range: [-1, 1]
  • tan(θ): Domain: all real numbers except θ = π/2 + nπ; Range: all real numbers
  • csc(θ): Domain: all real numbers except θ = nπ; Range: (-∞, -1] ∪ [1, ∞)
  • sec(θ): Domain: all real numbers except θ = π/2 + nπ; Range: (-∞, -1] ∪ [1, ∞)
  • cot(θ): Domain: all real numbers except θ = nπ; Range: all real numbers
Example 1: Evaluating Trigonometric Functions

Find the values of all six trigonometric functions for θ = π/4.

For θ = π/4, the point on the unit circle is (√2/2, √2/2).

sin(π/4) = √2/2 ≈ 0.7071

cos(π/4) = √2/2 ≈ 0.7071

tan(π/4) = sin(π/4) / cos(π/4) = (√2/2) / (√2/2) = 1

csc(π/4) = 1 / sin(π/4) = 1 / (√2/2) = √2 ≈ 1.4142

sec(π/4) = 1 / cos(π/4) = 1 / (√2/2) = √2 ≈ 1.4142

cot(π/4) = 1 / tan(π/4) = 1 / 1 = 1

Fundamental Trigonometric Identities

Trigonometric identities are equations involving trigonometric functions that are true for all values in the functions' domains. These fundamental identities form the basis for solving trigonometric problems.

Pythagorean Identities

sin²(θ) + cos²(θ) = 1
tan²(θ) + 1 = sec²(θ)
1 + cot²(θ) = csc²(θ)

Reciprocal Identities

sin(θ) = 1 / csc(θ)
cos(θ) = 1 / sec(θ)
tan(θ) = 1 / cot(θ)

Quotient Identities

tan(θ) = sin(θ) / cos(θ)
cot(θ) = cos(θ) / sin(θ)

Negative Angle Identities

sin(-θ) = -sin(θ)
cos(-θ) = cos(θ)
tan(-θ) = -tan(θ)
csc(-θ) = -csc(θ)
sec(-θ) = sec(θ)
cot(-θ) = -cot(θ)
Example 2: Using Fundamental Identities

If sin(θ) = 3/5 and θ is in the first quadrant, find cos(θ), tan(θ), and all other trigonometric functions.

Using the Pythagorean identity sin²(θ) + cos²(θ) = 1:

cos²(θ) = 1 - sin²(θ) = 1 - (3/5)² = 1 - 9/25 = 16/25

Since θ is in the first quadrant, cos(θ) is positive, so cos(θ) = 4/5

tan(θ) = sin(θ) / cos(θ) = (3/5) / (4/5) = 3/4

csc(θ) = 1 / sin(θ) = 1 / (3/5) = 5/3

sec(θ) = 1 / cos(θ) = 1 / (4/5) = 5/4

cot(θ) = 1 / tan(θ) = 1 / (3/4) = 4/3

Addition and Subtraction Formulas

These formulas allow us to find the trigonometric values of sums and differences of angles.

Sine Addition and Subtraction Formulas

sin(α + β) = sin(α)cos(β) + cos(α)sin(β)
sin(α - β) = sin(α)cos(β) - cos(α)sin(β)

Cosine Addition and Subtraction Formulas

cos(α + β) = cos(α)cos(β) - sin(α)sin(β)
cos(α - β) = cos(α)cos(β) + sin(α)sin(β)

Tangent Addition and Subtraction Formulas

tan(α + β) = [tan(α) + tan(β)] / [1 - tan(α)tan(β)]
tan(α - β) = [tan(α) - tan(β)] / [1 + tan(α)tan(β)]
Example 3: Using Addition Formulas

Find the exact value of sin(75°) using the addition formula.

We can express 75° as 45° + 30°, both of which have known trigonometric values.

Using the addition formula for sine:

sin(75°) = sin(45° + 30°) = sin(45°)cos(30°) + cos(45°)sin(30°)

sin(45°) = √2/2, cos(30°) = √3/2, cos(45°) = √2/2, sin(30°) = 1/2

sin(75°) = (√2/2)(√3/2) + (√2/2)(1/2) = (√6/4) + (√2/4) = (√6 + √2)/4

Double-Angle Formulas

Double-angle formulas express trigonometric functions of twice an angle in terms of the original angle.

sin(2θ) = 2sin(θ)cos(θ)

cos(2θ) = cos²(θ) - sin²(θ) = 2cos²(θ) - 1 = 1 - 2sin²(θ)

tan(2θ) = 2tan(θ) / [1 - tan²(θ)]

Power-Reduction Formulas

These are derived from double-angle formulas and are useful for integrating powers of trigonometric functions:

sin²(θ) = (1 - cos(2θ)) / 2
cos²(θ) = (1 + cos(2θ)) / 2
Example 4: Double-Angle Formula Application

Given cos(θ) = 3/5 and θ is in the first quadrant, find cos(2θ).

Using the double-angle formula cos(2θ) = 2cos²(θ) - 1:

cos(2θ) = 2(3/5)² - 1 = 2(9/25) - 1 = 18/25 - 1 = (18-25)/25 = -7/25

Note that 2θ is in the second quadrant where cosine is negative.

Half-Angle Formulas

Half-angle formulas express trigonometric functions of half an angle in terms of the original angle.

sin(θ/2) = ±√[(1 - cos(θ)) / 2]
cos(θ/2) = ±√[(1 + cos(θ)) / 2]
tan(θ/2) = (1 - cos(θ)) / sin(θ) = sin(θ) / (1 + cos(θ))

The sign depends on the quadrant in which θ/2 lies.

Example 5: Using Half-Angle Formulas

Find the exact value of cos(π/8) using the half-angle formula.

We can express π/8 as π/4 ÷ 2.

Using the half-angle formula for cosine:

cos(π/8) = cos(π/4 ÷ 2) = +√[(1 + cos(π/4)) / 2]

cos(π/4) = √2/2

cos(π/8) = +√[(1 + √2/2) / 2] = +√[(2 + √2) / 4] = +√(2 + √2) / 2

The positive square root is used because π/8 is in the first quadrant where cosine is positive.

Product-to-Sum and Sum-to-Product Formulas

Product-to-Sum Formulas

sin(α)sin(β) = [cos(α - β) - cos(α + β)] / 2
cos(α)cos(β) = [cos(α - β) + cos(α + β)] / 2
sin(α)cos(β) = [sin(α + β) + sin(α - β)] / 2
cos(α)sin(β) = [sin(α + β) - sin(α - β)] / 2

Sum-to-Product Formulas

sin(α) + sin(β) = 2sin((α + β)/2)cos((α - β)/2)
sin(α) - sin(β) = 2cos((α + β)/2)sin((α - β)/2)
cos(α) + cos(β) = 2cos((α + β)/2)cos((α - β)/2)
cos(α) - cos(β) = -2sin((α + β)/2)sin((α - β)/2)
Example 6: Product-to-Sum Conversion

Express sin(40°)sin(10°) as a sum or difference of cosines.

Using the product-to-sum formula sin(α)sin(β) = [cos(α - β) - cos(α + β)] / 2:

sin(40°)sin(10°) = [cos(40° - 10°) - cos(40° + 10°)] / 2

sin(40°)sin(10°) = [cos(30°) - cos(50°)] / 2

sin(40°)sin(10°) = [√3/2 - cos(50°)] / 2

Example 7: Evaluating Without a Calculator

Find the exact value of sin(150°) using reference angles and identities.

We can use the fact that 150° is in the second quadrant, where sine is positive.

The reference angle is 180° - 150° = 30°.

Therefore, sin(150°) = sin(180° - 30°) = sin(π - π/6)

Using the identity sin(π - θ) = sin(θ):

sin(150°) = sin(30°) = 1/2

Applications of Trigonometric Identities

Trigonometric identities have numerous applications in mathematics, physics, engineering, and other fields.

Simplifying Expressions

Identities can be used to simplify complex trigonometric expressions, making them easier to work with in calculus and other areas.

Proving Other Identities

Known identities can be used as building blocks to prove more complex identities.

Solving Equations

Trigonometric identities are essential for solving trigonometric equations, which appear in physics problems involving periodic phenomena.

Integrating Trigonometric Functions

Many integration techniques in calculus rely on trigonometric identities, especially substitution and integration by parts.

Example 8: Simplifying a Complex Expression

Simplify the expression: (sin(x) + cos(x))² - (sin(x) - cos(x))²

Let's expand both squared terms:

(sin(x) + cos(x))² = sin²(x) + 2sin(x)cos(x) + cos²(x)

(sin(x) - cos(x))² = sin²(x) - 2sin(x)cos(x) + cos²(x)

Subtracting: (sin(x) + cos(x))² - (sin(x) - cos(x))² = 4sin(x)cos(x)

Using the identity 2sin(x)cos(x) = sin(2x):

4sin(x)cos(x) = 2(2sin(x)cos(x)) = 2sin(2x)

Therefore, (sin(x) + cos(x))² - (sin(x) - cos(x))² = 2sin(2x)
