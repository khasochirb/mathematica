Fermat's and Euler's Theorems

Fundamental theorems in number theory with powerful applications in modern cryptography

Fermat's Little Theorem

Statement of the Theorem

Fermat's Little Theorem states that if p is a prime number and a is an integer not divisible by p, then:

ap-1 ≡ 1 (mod p)

An equivalent formulation that applies to all integers a (whether divisible by p or not) is:

ap ≡ a (mod p)
Example: Fermat's Little Theorem

Let's verify Fermat's Little Theorem for a = 3 and p = 5:

According to the theorem, 34 ≡ 1 (mod 5)

34 = 3 × 3 × 3 × 3 = 81

81 ÷ 5 = 16 remainder 1

So, 81 ≡ 1 (mod 5), confirming the theorem.

Using the alternative formulation:

35 ≡ 3 (mod 5)

35 = 3 × 34 = 3 × 81 = 243

243 ÷ 5 = 48 remainder 3

So, 243 ≡ 3 (mod 5), which also confirms the theorem.

Proof of Fermat's Little Theorem

There are several ways to prove Fermat's Little Theorem. Here's a proof using modular arithmetic and combinatorics:

Consider the set S = {1, 2, 3, ..., p-1} and the set T = {a, 2a, 3a, ..., (p-1)a}. We'll show that these sets are congruent modulo p.

  1. First, we need to show that all elements in T are distinct modulo p. If ra ≡ sa (mod p) for some 1 ≤ r, s ≤ p-1, then (r-s)a ≡ 0 (mod p). Since p is prime and a is not divisible by p, we must have r-s ≡ 0 (mod p). But since 1 ≤ r, s ≤ p-1, we have |r-s| < p, which means r = s. So, all elements in T are distinct modulo p.
  2. None of the elements in T is congruent to 0 modulo p, because a is not divisible by p.
  3. Therefore, T must be congruent to the set {1, 2, 3, ..., p-1} (i.e., S) modulo p, possibly in a different order.
  4. Multiplying all elements in S: 1 × 2 × ... × (p-1) = (p-1)!
  5. Multiplying all elements in T: a × 2a × ... × (p-1)a = ap-1 × (p-1)!
  6. Since S and T are congruent modulo p, their products are also congruent: (p-1)! ≡ ap-1 × (p-1)! (mod p)
  7. Since p is prime, (p-1)! is not divisible by p (by Wilson's theorem), so we can divide both sides by (p-1)! to get: 1 ≡ ap-1 (mod p)

Applications of Fermat's Little Theorem

  • Modular Exponentiation: Efficiently computing ab mod p by reducing the exponent modulo (p-1).
  • Primality Testing: The contrapositive of Fermat's Little Theorem provides a way to determine that a number is composite (the Fermat primality test).
  • Generating Cryptographic Keys: Used in various cryptographic protocols, including RSA encryption.

Euler's Totient Function

Definition and Properties

Euler's totient function, denoted φ(n), counts the positive integers up to n that are coprime to n. Two numbers are coprime (or relatively prime) if their greatest common divisor is 1.

Key properties of the totient function:

  • φ(1) = 1
  • If p is prime, then φ(p) = p - 1
  • If p is prime and k ≥ 1, then φ(pk) = pk - pk-1 = pk(1 - 1/p)
  • If m and n are coprime, then φ(mn) = φ(m) × φ(n) (multiplicative property)
  • For any n > 1, Σd|n φ(d) = n, where the sum is over all positive divisors d of n

Computing Euler's Totient Function

To compute φ(n) for a given n, we can use the prime factorization of n. If:

n = p₁k₁ × p₂k₂ × ... × prkr

Then:

φ(n) = n × (1 - 1/p₁) × (1 - 1/p₂) × ... × (1 - 1/pr)

Or equivalently:

φ(n) = n × ∏p|n (1 - 1/p)
Example: Computing Euler's Totient Function

Calculate φ(36):

First, find the prime factorization of 36: 36 = 22 × 32

Using the formula:

φ(36) = 36 × (1 - 1/2) × (1 - 1/3) = 36 × (1/2) × (2/3) = 12

Alternatively, we can verify this directly by counting the numbers from 1 to 36 that are coprime to 36:

1, 5, 7, 11, 13, 17, 19, 23, 25, 29, 31, 35

There are 12 such numbers, confirming that φ(36) = 12.

Euler's Theorem

Statement of the Theorem

Euler's Theorem (also known as Euler's Totient Theorem) generalizes Fermat's Little Theorem to composite moduli. It states that if a and n are coprime positive integers, then:

aφ(n) ≡ 1 (mod n)

where φ(n) is Euler's totient function.

Example: Euler's Theorem

Verify Euler's Theorem for a = 5 and n = 9:

First, determine φ(9): 9 = 32, so φ(9) = 9 × (1 - 1/3) = 9 × (2/3) = 6

According to Euler's Theorem: 56 ≡ 1 (mod 9)

Calculate 56: 56 = 15625

15625 ÷ 9 = 1736 remainder 1

So, 56 ≡ 1 (mod 9), confirming Euler's Theorem.

Proof of Euler's Theorem

The proof of Euler's Theorem is similar to the proof of Fermat's Little Theorem, but it uses the concept of reduced residue system:

  1. Consider the reduced residue system modulo n: the set of φ(n) integers from 1 to n that are coprime to n. Let's call this set R = {r₁, r₂, ..., rφ(n)}.
  2. Multiply each element in R by a (which is coprime to n) and take the result modulo n. Let's call this set S = {ar₁ mod n, ar₂ mod n, ..., arφ(n) mod n}.
  3. It can be shown that S is also a reduced residue system modulo n, i.e., S is a permutation of R.
  4. Multiplying all elements in R: r₁ × r₂ × ... × rφ(n)
  5. Multiplying all elements in S: ar₁ × ar₂ × ... × arφ(n) = aφ(n) × r₁ × r₂ × ... × rφ(n)
  6. Since S is a permutation of R modulo n, their products are congruent modulo n: r₁ × r₂ × ... × rφ(n) ≡ aφ(n) × r₁ × r₂ × ... × rφ(n) (mod n)
  7. Since all elements in R are coprime to n, their product is also coprime to n. Thus, we can divide both sides by r₁ × r₂ × ... × rφ(n) to get: 1 ≡ aφ(n) (mod n)

Relation to Fermat's Little Theorem

Euler's Theorem is a generalization of Fermat's Little Theorem. When n is a prime number p, φ(p) = p - 1, and Euler's Theorem becomes:

ap-1 ≡ 1 (mod p)

which is precisely Fermat's Little Theorem.

Applications in Cryptography

RSA Encryption

RSA (Rivest-Shamir-Adleman) is one of the first public-key cryptosystems, and it relies directly on Euler's Theorem. The security of RSA is based on the practical difficulty of factoring the product of two large prime numbers.

The RSA algorithm works as follows:

  1. Choose two distinct large prime numbers p and q.
  2. Compute n = p × q.
  3. Compute φ(n) = (p - 1) × (q - 1).
  4. Choose an integer e such that 1 < e < φ(n) and gcd(e, φ(n)) = 1.
  5. Compute d such that d × e ≡ 1 (mod φ(n)).
  6. The public key is (n, e), and the private key is (n, d).
  7. To encrypt a message m, compute c = me mod n.
  8. To decrypt the ciphertext c, compute m = cd mod n.

The correctness of RSA decryption is based on Euler's Theorem: for any message m coprime to n, we have mφ(n) ≡ 1 (mod n). Since d × e ≡ 1 (mod φ(n)), we have d × e = k × φ(n) + 1 for some integer k. Therefore:

cd = (me)d = me×d = mk×φ(n)+1 = m × (mφ(n))k ≡ m × 1k ≡ m (mod n)
Example: RSA Encryption and Decryption

Let's work through a simple example with small numbers:

  1. Choose p = 5 and q = 11.
  2. Compute n = 5 × 11 = 55.
  3. Compute φ(n) = (5 - 1) × (11 - 1) = 4 × 10 = 40.
  4. Choose e = 7 (coprime to 40).
  5. Compute d such that 7d ≡ 1 (mod 40). Using the Extended Euclidean Algorithm, we get d = 23.
  6. Public key: (55, 7), Private key: (55, 23).
  7. To encrypt a message m = 13, compute c = 137 mod 55 = 8.
  8. To decrypt the ciphertext c = 8, compute m = 823 mod 55 = 13.

Primality Testing

Fermat's Little Theorem is the basis for the Fermat primality test. If we find an integer a such that an-1 ≢ 1 (mod n), then n is definitely composite. However, if an-1 ≡ 1 (mod n) for many values of a, then n is likely (but not guaranteed) to be prime.

Composite numbers that satisfy an-1 ≡ 1 (mod n) for many values of a are called Carmichael numbers or Fermat pseudoprimes. To handle such numbers, more sophisticated primality tests like the Miller-Rabin test incorporate additional checks.

Extensions and Related Theorems

Wilson's Theorem

Wilson's Theorem states that a positive integer n > 1 is prime if and only if:

(n-1)! ≡ -1 (mod n)

This theorem provides a necessary and sufficient condition for primality, but it's not practical for primality testing due to the computational difficulty of calculating factorials for large numbers.

Chinese Remainder Theorem

The Chinese Remainder Theorem describes how to reconstruct a number from its remainders modulo a set of pairwise coprime moduli. It states that if m₁, m₂, ..., mk are pairwise coprime positive integers, then the system of congruences:

x ≡ a₁ (mod m₁)
x ≡ a₂ (mod m₂)
...
x ≡ ak (mod mk)

has a unique solution modulo M = m₁ × m₂ × ... × mk.

Carmichael's Theorem

Carmichael's Theorem provides a refinement of Euler's Theorem. It states that for any coprime integers a and n, there exists a number λ(n) (known as the Carmichael function) such that:

aλ(n) ≡ 1 (mod n)

Furthermore, λ(n) is the smallest positive integer with this property.

The Carmichael function λ(n) is always a divisor of φ(n), and in many cases, it's significantly smaller, making computations more efficient.
