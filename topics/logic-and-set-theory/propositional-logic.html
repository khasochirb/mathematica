Introduction to Propositional Logic

Propositional logic is the study of propositions, statements that are either true or false but not both. It forms the foundation of mathematical reasoning and provides the basic building blocks for constructing valid arguments.

In propositional logic, we use variables (often p, q, r, ...) to represent simple propositions, and we combine them using logical connectives to form compound propositions.

Basic Logical Connectives

  • Negation (¬): The statement "not p" is true when p is false, and false when p is true.
  • Conjunction (∧): The statement "p and q" is true only when both p and q are true.
  • Disjunction (∨): The statement "p or q" is true when at least one of p or q is true.
  • Implication (→): The statement "if p then q" is false only when p is true and q is false.
  • Biconditional (↔): The statement "p if and only if q" is true when p and q have the same truth value.

Logical Equivalences and Tautologies

Two propositional formulas are logically equivalent if they have the same truth value for all possible assignments of truth values to their variables.

A tautology is a formula that is true for all possible truth value assignments.

Example: De Morgan's Laws

De Morgan's laws are important logical equivalences:

  • ¬(p ∧ q) ≡ (¬p) ∨ (¬q)
  • ¬(p ∨ q) ≡ (¬p) ∧ (¬q)

These laws describe how negation distributes over conjunction and disjunction.

Some important tautologies include:

  • Law of Excluded Middle: p ∨ ¬p
  • Law of Non-Contradiction: ¬(p ∧ ¬p)
  • Double Negation: ¬¬p ↔ p

Applications and Further Reading

Propositional logic is fundamental to many areas of mathematics and computer science:

  • Mathematical proofs and formal verification
  • Boolean algebra and digital circuit design
  • Computer programming and algorithm design
  • Database query languages

To deepen your understanding of propositional logic, you may want to explore:

  • Truth tables and logical equivalence
  • Formal proof systems (natural deduction, sequent calculus)
  • Boolean algebra and normal forms
  • Predicate logic, which extends propositional logic
