Introduction to Mathematical Proofs

A mathematical proof is a rigorous argument that demonstrates the truth of a mathematical statement. Proofs are the foundation of mathematical knowledge, providing certainty about mathematical truths and insights into the structure of mathematical theories.

Unlike in other sciences where theories are validated through experimental evidence, mathematics relies on logical deduction from axioms and previously established results. This deductive approach gives mathematics its unique character of absolute certainty.

The Structure of a Proof

A typical proof consists of:

  • Statement: The proposition to be proven, often in the form of "if P, then Q" (implication).
  • Given Information: The assumptions or conditions that are known to be true.
  • Chain of Reasoning: Logical steps connecting the given information to the conclusion.
  • Conclusion: The final result that confirms the statement is true.

Proof and Mathematical Rigor

Mathematical rigor refers to the precision and thoroughness required in proofs. A rigorous proof:

  • Starts from clearly stated assumptions
  • Follows logically valid steps
  • Considers all possible cases
  • Makes no unstated assumptions
  • Contains no logical fallacies

Basic Methods of Proof

Direct Proof

A direct proof starts with the hypotheses and, through a sequence of logical deductions, arrives at the conclusion. This method is often used to prove statements of the form "if P, then Q."

Example: Direct Proof

Theorem: If n is an even integer, then n² is even.

Proof:

1. Since n is even, we can write n = 2k for some integer k.

2. Then n² = (2k)² = 4k² = 2(2k²).

3. Since 2k² is an integer, n² = 2(2k²) is even by definition.

4. Therefore, if n is even, then n² is even.

Proof by Contraposition

A proof by contraposition uses the logical equivalence between "if P, then Q" and "if not Q, then not P." Instead of proving the original statement, we prove its contrapositive.

Example: Proof by Contraposition

Theorem: If n² is odd, then n is odd.

Proof:

We'll prove the contrapositive: "If n is not odd (i.e., n is even), then n² is not odd (i.e., n² is even)."

1. Assume n is even, so n = 2k for some integer k.

2. Then n² = (2k)² = 4k² = 2(2k²).

3. Since 2k² is an integer, n² = 2(2k²) is even.

4. Therefore, if n is even, then n² is even (or equivalently, if n² is odd, then n is odd).

Proof by Contradiction

A proof by contradiction assumes the statement to be proven is false, and shows that this assumption leads to a logical contradiction. This implies the statement must be true.

Example: Proof by Contradiction

Theorem: √2 is irrational.

Proof:

1. Assume, for contradiction, that √2 is rational.

2. Then √2 = a/b for some integers a and b (b ≠ 0) with no common factors.

3. Squaring both sides: 2 = a²/b²

4. Multiplying both sides by b²: 2b² = a²

5. Since the left side is even, a² must be even, which means a must be even (we showed earlier that if n² is even, then n is even).

6. So a = 2k for some integer k.

7. Substituting: 2b² = (2k)² = 4k²

8. Therefore, b² = 2k²

9. By the same reasoning, b² is even, so b is even.

10. But this contradicts our assumption that a and b have no common factors (since they're both even).

11. Therefore, our assumption that √2 is rational must be false, so √2 is irrational.

Proof by Induction

Mathematical induction is used to prove that a statement is true for all natural numbers or integers starting from some base case. It involves two steps:

  1. Base Case: Prove the statement is true for the smallest value (often n = 1 or n = 0).
  2. Inductive Step: Assume the statement is true for some arbitrary k, and prove it's true for k+1.
Example: Proof by Induction

Theorem: For all natural numbers n, 1 + 2 + 3 + ... + n = n(n+1)/2.

Base Case (n = 1):

Left side: 1

Right side: 1(1+1)/2 = 1

The formula holds for n = 1.

Inductive Step:

Assume the formula holds for n = k: 1 + 2 + ... + k = k(k+1)/2

We need to prove it holds for n = k+1: 1 + 2 + ... + k + (k+1) = (k+1)(k+2)/2

Starting with the left side:

1 + 2 + ... + k + (k+1) = [1 + 2 + ... + k] + (k+1) = k(k+1)/2 + (k+1)

= (k+1)[k/2 + 1] = (k+1)[k/2 + 2/2] = (k+1)(k+2)/2

This equals the right side, so the formula holds for n = k+1.

By the principle of mathematical induction, the formula holds for all natural numbers n.

Advanced Proof Techniques

Proof by Cases

A proof by cases divides the problem into several exhaustive cases and proves each case separately. This technique is useful when different scenarios require different approaches.

Example: Proof by Cases

Theorem: For any integer n, the value of n² - n + 41 is prime for n = 1, 2, ..., 40.

Proof:

We need to check each case n = 1, 2, ..., 40 separately:

For n = 1: 1² - 1 + 41 = 41, which is prime.

For n = 2: 2² - 2 + 41 = 43, which is prime.

...

For n = 40: 40² - 40 + 41 = 1601, which is prime.

Note: We could verify each case explicitly or use computational tools.

Proof by Counterexample

A proof by counterexample disproves a universal statement ("For all x, P(x)") by finding a specific example where the statement is false.

Example: Proof by Counterexample

Statement: For all integers n, the polynomial n² - n + 41 always produces a prime number.

Counterexample:

Let's try n = 41:

41² - 41 + 41 = 1681 = 41²

Since 1681 = 41² is not prime, the statement is false.

Existence and Uniqueness Proofs

Some theorems assert the existence or uniqueness of a mathematical object. These proofs often require special approaches:

  • Existence Proof: Show that at least one object with the required properties exists.
  • Uniqueness Proof: Show that there cannot be more than one such object.
Example: Existence and Uniqueness Proof

Theorem: For any real numbers a, b, and c, where a ≠ 0, the quadratic equation ax² + bx + c = 0 has exactly one solution if and only if b² = 4ac.

Proof:

Existence: Using the quadratic formula, x = (-b ± √(b² - 4ac))/(2a)

When b² = 4ac, the discriminant is zero, so x = -b/(2a)

Since a ≠ 0, this solution exists.

Uniqueness: When b² = 4ac, the two solutions given by the quadratic formula x = (-b ± √(b² - 4ac))/(2a) coincide as x = -b/(2a).

Conversely, if there is exactly one solution, then the discriminant must be zero (b² - 4ac = 0), which means b² = 4ac.

Common Proof Strategies and Hints

Working Backward

Sometimes it's helpful to start from the conclusion and work backward to the given conditions. This can provide insights into the forward proof path.

Introducing Auxiliary Elements

Creating additional constructions, variables, or objects can simplify complex proofs.

Proof by Minimum Counterexample

When trying to prove a statement for all natural numbers, assume there's a counterexample and consider the smallest one. This often leads to a contradiction.

Proof by Symmetry or Invariance

Identifying symmetries or invariant properties can simplify proofs by reducing the number of cases to consider.

Example: Using Symmetry

Theorem: In a triangle, if two sides are equal, then the angles opposite these sides are equal.

Proof:

Consider a triangle ABC where AB = AC.

The line from A to the midpoint of BC creates two triangles.

By the properties of congruent triangles (Side-Side-Side congruence), these two triangles are congruent.

Therefore, the angles opposite the equal sides must be equal: ∠B = ∠C.

Fallacies and Common Errors in Proofs

Circular Reasoning (Begging the Question)

This fallacy occurs when the conclusion is assumed, directly or indirectly, in the premises of the argument.

Affirming the Consequent

The logical fallacy of concluding P from "P implies Q" and Q. For example: "If it rains, the ground is wet. The ground is wet, so it must have rained."

Misapplying Proof Techniques

Using a proof technique inappropriately, such as using induction without proving the base case.

Overlooking Special Cases

Failing to consider all possible scenarios, especially edge cases or degenerate cases.

Example: Spotting a Fallacy

Flawed "Proof": All horses are the same color.

Proposed Induction Proof:

Base case (n = 1): In a set of 1 horse, all horses are the same color. (True)

Inductive step: Assume that in any set of k horses, all horses are the same color.

Consider a set of k+1 horses. Remove one horse to get a set of k horses, which all have the same color by the inductive hypothesis.

Now reinsert the removed horse and remove a different horse. The resulting set of k horses again all have the same color.

Therefore, all k+1 horses must have the same color.

The Flaw: The induction breaks down at the step from n = 1 to n = 2. With two horses, removing one and then the other doesn't provide a common set to establish the same color.

Writing and Reading Proofs

Elements of a Well-Written Proof

  • Clarity: Each step should be clearly stated and justified.
  • Organization: The proof should follow a logical structure.
  • Economy: Include necessary details, but avoid excessive verbosity.
  • Motivation: When appropriate, explain the intuition behind key steps.

Reading and Understanding Proofs

  • Active Reading: Try to anticipate the next step or fill in missing details.
  • Identify Structure: Recognize the proof technique being used.
  • Focus on Transitions: Pay attention to how different parts of the proof connect.
  • Check Completeness: Ensure all cases are covered and all claims are justified.
