Predicate Logic

Understanding quantifiers and predicates in mathematical reasoning

Introduction to Predicate Logic

Predicate logic, also known as first-order logic, is a powerful extension of propositional logic that allows us to express more complex statements and relationships. While propositional logic deals with simple statements that are either true or false, predicate logic introduces variables, quantifiers, and predicates to express properties of objects and relationships between them.

This enhanced expressiveness makes predicate logic an essential foundation for mathematics, computer science, linguistics, and philosophy. It provides the formal framework necessary for precisely stating mathematical theorems, defining mathematical structures, and constructing rigorous proofs.

From Propositional to Predicate Logic

Consider the following statements:

  • "All natural numbers are positive."
  • "Some integers are negative."
  • "If a number is prime, then it has exactly two distinct divisors."

Propositional logic can't adequately represent these statements because they involve properties that apply to collections of objects, and relationships between different properties. Predicate logic adds the necessary tools to formalize such statements.

Predicates and Quantifiers

Predicates

A predicate is a statement that contains variables and becomes a proposition when specific values are assigned to those variables. Predicates are typically denoted by capital letters, with variables as arguments.

For example, let P(x) represent "x is a prime number" where x is a variable that can be any integer. Then:

  • P(2) is true (2 is prime)
  • P(4) is false (4 is not prime)
  • P(7) is true (7 is prime)

We can also have predicates with multiple variables. Let R(x, y) represent "x is greater than y". Then:

  • R(5, 3) is true (5 > 3)
  • R(2, 7) is false (2 is not > 7)

Quantifiers

Quantifiers allow us to express statements about collections of objects. The two main quantifiers are:

Universal Quantifier (∀): "For all" or "For every"
Existential Quantifier (∃): "There exists" or "For some"

Using our previous examples:

  • "All natural numbers are positive" can be written as: ∀x(N(x) → P(x))
    where N(x) means "x is a natural number" and P(x) means "x is positive".
  • "Some integers are negative" can be written as: ∃x(I(x) ∧ N(x))
    where I(x) means "x is an integer" and N(x) means "x is negative".
Example: Translating Between English and Predicate Logic

Let's translate the following statements to and from predicate logic:

1. "Every prime number greater than 2 is odd."

Let P(x) mean "x is prime", G(x, y) mean "x is greater than y", and O(x) mean "x is odd".

Translation: ∀x((P(x) ∧ G(x, 2)) → O(x))

2. ∃x(S(x) ∧ ∀y(S(y) → (x ≤ y)))

Let S(x) mean "x is a student" and ≤ represent "is younger than or the same age as".

Translation: "There exists a student who is younger than or the same age as all other students."

In other words, "There is a youngest student."

Formal Syntax and Semantics

Syntax of Predicate Logic

The formal language of predicate logic consists of:

  • Variables: Typically denoted by lowercase letters (x, y, z, etc.)
  • Constants: Specific objects, often denoted by lowercase letters (a, b, c, etc.)
  • Function symbols: Represent functions from objects to objects (f, g, h, etc.)
  • Predicate symbols: Represent properties or relations (P, Q, R, etc.)
  • Logical connectives: ¬ (not), ∧ (and), ∨ (or), → (implies), ↔ (if and only if)
  • Quantifiers: ∀ (for all), ∃ (there exists)
  • Parentheses: Used for grouping

Terms and Formulas

Terms represent objects and are built from:

  • Variables
  • Constants
  • Functions applied to other terms

Formulas (or well-formed formulas, WFFs) are constructed according to the following rules:

  • If P is an n-ary predicate symbol and t₁, ..., tₙ are terms, then P(t₁, ..., tₙ) is a formula.
  • If φ is a formula, then ¬φ is a formula.
  • If φ and ψ are formulas, then (φ ∧ ψ), (φ ∨ ψ), (φ → ψ), and (φ ↔ ψ) are formulas.
  • If φ is a formula and x is a variable, then ∀x φ and ∃x φ are formulas.

Semantics: Interpretation and Models

The meaning of formulas in predicate logic is given by interpretations or structures, which consist of:

  • A non-empty set D (the domain or universe)
  • For each constant symbol, an element of D
  • For each n-ary function symbol, a function from Dⁿ to D
  • For each n-ary predicate symbol, a subset of Dⁿ (representing the tuples for which the predicate is true)

A formula is satisfiable if there exists an interpretation under which it is true. A formula is valid (or a tautology) if it is true under all interpretations.

Example: Interpretation in a Domain

Consider the formula: ∀x∃y R(x, y)

Let's interpret this in two different domains:

1. Domain: Set of integers, R(x, y) means "y > x"

This formula is true in this interpretation because for any integer x, we can always find another integer y that is greater than x (e.g., y = x + 1).

2. Domain: Set of natural numbers, R(x, y) means "y divides x"

This formula is true in this interpretation because for any natural number x, we can always find y = 1, which divides every natural number.

3. Domain: Set of natural numbers, R(x, y) means "x divides y"

This formula is false in this interpretation. For example, if x = 4, there is no natural number y such that 4 divides y, other than multiples of 4. So for x = 4, we would need ∃y such that 4 divides y, which means y must be 4, 8, 12, etc. But this doesn't work for all x (consider x = 0).

Negation of Quantified Statements

Understanding how to negate quantified statements is crucial for many mathematical proofs, especially when using proof by contradiction.

Negation Rules

¬(∀x P(x)) ≡ ∃x ¬P(x)

¬(∃x P(x)) ≡ ∀x ¬P(x)

In words:

  • "It is not the case that all x have property P" is equivalent to "There exists an x that does not have property P."
  • "It is not the case that there exists an x with property P" is equivalent to "All x do not have property P."

Multiple Quantifiers

For statements with multiple quantifiers, negation is more complex. We negate the entire statement and then apply the negation rules to each quantifier from left to right, changing ∀ to ∃ and vice versa, and negating the matrix (the part after the quantifiers).

Example: Negating Multiple Quantifiers

Let's negate the statement: ∀x∃y(x + y = 0)

Step 1: Negate the entire statement: ¬(∀x∃y(x + y = 0))

Step 2: Apply the negation rule for ∀: ∃x¬(∃y(x + y = 0))

Step 3: Apply the negation rule for ∃: ∃x∀y¬(x + y = 0)

The final negation is: ∃x∀y(x + y ≠ 0)

In plain English, the original statement means "For every number x, there exists a number y such that x + y = 0" (which is true in the domain of real numbers, since y = -x works).

The negation means "There exists a number x such that for all numbers y, x + y ≠ 0" (which is false in the domain of real numbers, since for any x, we can choose y = -x to make x + y = 0).

Predicate Logic and Mathematical Proofs

Inference Rules for Predicate Logic

In addition to the inference rules from propositional logic, predicate logic has rules specifically for quantifiers:

Universal Instantiation (UI): From ∀x P(x), we can infer P(c) for any specific constant c in the domain.

Existential Generalization (EG): From P(c) for some specific constant c, we can infer ∃x P(x).

Universal Generalization (UG): If we can prove P(c) for an arbitrary constant c (with no special assumptions about c), then we can infer ∀x P(x).

Existential Instantiation (EI): From ∃x P(x), we can infer P(c) for some new constant c that doesn't appear elsewhere in the proof.

Proof Techniques Using Predicate Logic

Direct Proof for Universal Statements: To prove ∀x P(x), assume x is arbitrary and prove P(x).

Proof by Counterexample: To disprove ∀x P(x), find a specific value of x for which P(x) is false.

Constructive Proof for Existential Statements: To prove ∃x P(x), find a specific value of x for which P(x) is true.

Proof by Contradiction: To prove a statement S, assume ¬S and derive a contradiction.

Example: A Simple Proof Using Predicate Logic

Let's prove that if n is an odd integer, then n² is also odd.

This can be formalized as: ∀n(Odd(n) → Odd(n²))

Proof:

1. Let n be an arbitrary integer such that Odd(n).

2. By definition, Odd(n) means n = 2k + 1 for some integer k.

3. Therefore, n² = (2k + 1)² = 4k² + 4k + 1 = 2(2k² + 2k) + 1.

4. Let m = 2k² + 2k, which is an integer since k is an integer.

5. Then n² = 2m + 1, which means n² is odd by definition.

6. Since n was arbitrary, we have proven that for all integers n, if n is odd, then n² is odd.

Applications and Extensions

Applications in Mathematics

Predicate logic is fundamental to various areas of mathematics:

  • Set Theory: Axioms of set theory are formulated in predicate logic.
  • Analysis: Definitions of limits, continuity, and derivatives use quantifiers extensively.
  • Abstract Algebra: Definitions of algebraic structures like groups and rings use predicate logic.
  • Number Theory: Statements about properties of numbers are expressed using quantifiers.

Applications in Computer Science

Predicate logic has numerous applications in computer science:

  • Logic Programming: Languages like Prolog are based on a subset of predicate logic.
  • Database Query Languages: SQL and relational algebra are closely related to predicate logic.
  • Formal Verification: Proving correctness of algorithms and systems uses predicate logic.
  • Artificial Intelligence: Knowledge representation and automated reasoning systems use predicate logic.

Extensions of Predicate Logic

First-order predicate logic can be extended in various ways:

  • Second-Order Logic: Allows quantification over predicates and functions, not just variables.
  • Modal Logic: Adds operators for "necessity" (□) and "possibility" (◇).
  • Temporal Logic: Adds operators for reasoning about time.
  • Fuzzy Logic: Allows truth values between 0 and 1, not just true and false.
Example: Formalizing Mathematical Definitions

Here are some common mathematical concepts expressed in predicate logic:

Injectivity (one-to-one): A function f is injective if ∀x∀y(f(x) = f(y) → x = y).

Surjectivity (onto): A function f from A to B is surjective if ∀y∈B∃x∈A(f(x) = y).

Limit of a sequence: L is the limit of sequence (aₙ) if:

∀ε > 0 ∃N ∀n > N (|aₙ - L| < ε)

Continuity of a function: A function f is continuous at a point c if:

∀ε > 0 ∃δ > 0 ∀x (|x - c| < δ → |f(x) - f(c)| < ε)
