Linear Transformations

Understanding the geometric and algebraic properties of linear transformations

Home » Resources » Mathematics Roadmap » Matrices and Determinants » Linear Transformations

Introduction to Linear Transformations

A linear transformation is a function between vector spaces that preserves vector addition and scalar multiplication. In simpler terms, it's a way of transforming one vector space into another while maintaining the "linear structure" of the space.

Definition

A function T: V → W between vector spaces V and W is a linear transformation if for all vectors u, v ∈ V and all scalars c:

  1. Additivity: T(u + v) = T(u) + T(v)
  2. Homogeneity: T(cu) = cT(u)

Matrix Representation

Every linear transformation between finite-dimensional vector spaces can be represented by a matrix. If T: ℝn → ℝm is a linear transformation, then there exists a unique m × n matrix A such that for any vector x ∈ ℝn:

T(x) = Ax
Example: Linear Transformation

Consider the linear transformation T: ℝ2 → ℝ2 defined by:

T([x, y]) = [2x + y, x - y]

We can represent this transformation as a matrix multiplication:

T([x, y]) = [
  2  1
  1  -1
] [
  x
  y
]

To verify this is a linear transformation, we check:

1. Additivity: T(u + v) = T(u) + T(v)

2. Homogeneity: T(cu) = cT(u)

Both properties hold because matrix multiplication satisfies these properties.

Types of Linear Transformations

Scaling

A scaling transformation changes the size of vectors by multiplying each component by a scalar. In 2D, a scaling transformation with factors sx and sy is represented by the matrix:

S = [
  sx  0
  0  sy
]
Example: Scaling

The transformation that doubles coordinates in the x-direction and triples them in the y-direction is represented by:

S = [
  2  0
  0  3
]

For a vector [1, 1], the result is:

S[1, 1] = [
  2  0
  0  3
] [
  1
  1
] = [
  2
  3
]

Rotation

A rotation transformation in 2D by angle θ (counter-clockwise) is represented by the matrix:

R = [
  cos(θ)  -sin(θ)
  sin(θ)  cos(θ)
]
Example: Rotation

The transformation that rotates vectors by 90° counter-clockwise is represented by:

R = [
  cos(90°)  -sin(90°)
  sin(90°)  cos(90°)
] = [
  0  -1
  1  0
]

For a vector [1, 0], the result is:

R[1, 0] = [
  0  -1
  1  0
] [
  1
  0
] = [
  0
  1
]

This rotates the vector [1, 0] to [0, 1], which is a 90° counter-clockwise rotation.

Reflection

A reflection transformation flips vectors across a line or plane. In 2D, a reflection across the x-axis is represented by:

Rx = [
  1  0
  0  -1
]

And a reflection across the y-axis is represented by:

Ry = [
  -1  0
  0  1
]

Shear

A shear transformation slants the shape of an object. The x-shear and y-shear matrices in 2D are:

Sx = [
  1  k
  0  1
]
Sy = [
  1  0
  k  1
]

where k is the shear factor.

Example: Shear

A horizontal shear with factor 2 is represented by:

Sx = [
  1  2
  0  1
]

For a vector [1, 1], the result is:

Sx[1, 1] = [
  1  2
  0  1
] [
  1
  1
] = [
  3
  1
]

This transforms [1, 1] to [3, 1], shearing the vector horizontally.

Projection

A projection transformation projects vectors onto a subspace. For example, the projection onto the x-axis in 2D is represented by:

Px = [
  1  0
  0  0
]

And the projection onto the y-axis is represented by:

Py = [
  0  0
  0  1
]

Properties of Linear Transformations

Kernel and Image

The kernel (or null space) of a linear transformation T: V → W is the set of all vectors in V that T maps to the zero vector in W:

ker(T) = {v ∈ V | T(v) = 0}

The image (or range) of T is the set of all vectors in W that are the result of applying T to some vector in V:

im(T) = {T(v) | v ∈ V}
Example: Kernel and Image

For the linear transformation T: ℝ2 → ℝ2 defined by:

T([x, y]) = [x + y, x + y]

The kernel consists of all vectors [x, y] such that [x + y, x + y] = [0, 0], which means x + y = 0. So:

ker(T) = {[x, -x] | x ∈ ℝ}

The image consists of all vectors of the form [z, z] where z = x + y for some x, y ∈ ℝ. So:

im(T) = {[z, z] | z ∈ ℝ}

Rank and Nullity

The rank of a linear transformation is the dimension of its image:

rank(T) = dim(im(T))

The nullity of a linear transformation is the dimension of its kernel:

nullity(T) = dim(ker(T))

The rank-nullity theorem states that for a linear transformation T: V → W between finite-dimensional vector spaces:

dim(V) = rank(T) + nullity(T)

Invertibility

A linear transformation T: V → W is invertible if there exists a linear transformation S: W → V such that S(T(v)) = v for all v ∈ V and T(S(w)) = w for all w ∈ W.

A linear transformation is invertible if and only if it is bijective (both injective and surjective). For a transformation represented by a square matrix A, this is equivalent to A being invertible (det(A) ≠ 0).

Composition

The composition of two linear transformations is also a linear transformation. If T: U → V and S: V → W are linear transformations, then their composition S ∘ T: U → W is defined by (S ∘ T)(u) = S(T(u)) for all u ∈ U.

If T and S are represented by matrices A and B respectively, then S ∘ T is represented by the matrix product BA.

Eigenvalues and Eigenvectors in Transformations

Eigenvalues and eigenvectors provide important information about the behavior of linear transformations.

Basic Concepts

For a linear transformation T: V → V, a non-zero vector v ∈ V is an eigenvector of T if T(v) is a scalar multiple of v. That is, there exists a scalar λ such that:

T(v) = λv

The scalar λ is called an eigenvalue of T corresponding to the eigenvector v.

If T is represented by a matrix A, then the eigenvalues of T are the solutions to the characteristic equation:

det(A - λI) = 0

where I is the identity matrix of the same size as A.

Example: Finding Eigenvalues and Eigenvectors

Consider the linear transformation T: ℝ2 → ℝ2 represented by the matrix:

A = [
  3  1
  1  3
]

To find the eigenvalues, we solve det(A - λI) = 0:

det([
  3 - λ  1
  1  3 - λ
]) = 0
(3 - λ)(3 - λ) - 1 = 0
(3 - λ)² - 1 = 0
(3 - λ)² = 1
3 - λ = ±1
λ = 2 or λ = 4

For λ = 2, we find the eigenvector v by solving (A - 2I)v = 0:

[
  1  1
  1  1
] [
  v1
  v2
] = [
  0
  0
]

This gives v1 = -v2, so an eigenvector for λ = 2 is [1, -1].

For λ = 4, we solve (A - 4I)v = 0:

[
  -1  1
  1  -1
] [
  v1
  v2
] = [
  0
  0
]

This gives v1 = v2, so an eigenvector for λ = 4 is [1, 1].

Geometric Interpretation

Eigenvectors represent directions in which the linear transformation acts as a simple scaling. If v is an eigenvector of T with eigenvalue λ, then applying T to v scales v by a factor of λ without changing its direction (except possibly reversing it if λ < 0).

This property is particularly useful in understanding the behavior of iterative applications of a transformation, as in dynamical systems and Markov chains.

Applications of Linear Transformations

Computer Graphics

Linear transformations are fundamental in computer graphics for operations such as scaling, rotation, shearing, and projection of objects. These transformations can be efficiently composed by multiplying their corresponding matrices.

Data Analysis

In statistics and machine learning, linear transformations are used for dimensionality reduction (e.g., Principal Component Analysis), feature extraction, and data normalization.

Physics

Linear transformations model various physical phenomena, such as reflections and rotations in optics, coordinate transformations in relativity, and transformations between different reference frames in mechanics.

Signal Processing

The Fourier transform, a fundamental tool in signal processing, is a linear transformation that converts a function of time to a function of frequency.

Engineering

In control theory, linear transformations are used to analyze the behavior of linear systems and to design control systems. In electrical engineering, they are used to model circuit behavior.

Example: Image Transformation

In image processing, a grayscale image can be represented as a matrix of pixel intensities. Applying a linear transformation to this matrix can produce various effects:

  • Scaling changes the size of the image.
  • Rotation rotates the image around a specified point.
  • Shearing slants the image along a specified axis.
  • Reflection flips the image across a specified axis.

For example, to rotate an image by 180°, you would apply the transformation matrix:

R = [
  -1  0
  0  -1
]

This transforms each pixel (x, y) to (-x, -y), effectively rotating the image by 180°.
