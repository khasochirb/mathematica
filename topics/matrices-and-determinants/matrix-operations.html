Matrix Operations

Fundamental operations and properties of matrices in linear algebra

Home » Resources » Mathematics Roadmap » Matrices and Determinants » Matrix Operations

Introduction to Matrices

A matrix is a rectangular array of numbers, symbols, or expressions, arranged in rows and columns. Matrices provide a compact way to represent and manipulate systems of linear equations, linear transformations, and many other mathematical structures.

Basic Terminology

  • Dimension: An m × n matrix has m rows and n columns.
  • Square Matrix: A matrix with the same number of rows and columns (m = n).
  • Element: Each entry in a matrix, denoted as aij, where i represents the row and j represents the column.
  • Row/Column Vector: A matrix with only one row or one column.
Example: Matrix Representation

A 2 × 3 matrix A:

A = [
  2  4  6
  1  3  5
]

Here, a11 = 2, a12 = 4, a13 = 6, a21 = 1, a22 = 3, and a23 = 5.

Matrix Addition and Subtraction

Matrix addition and subtraction are performed element by element. Two matrices can be added or subtracted only if they have the same dimensions.

Matrix Addition

For matrices A and B of the same dimension, the sum A + B is a matrix C where cij = aij + bij for all i, j.

Example: Matrix Addition

For:

A = [
  1  2
  3  4
]

and

B = [
  5  6
  7  8
]

The sum A + B is:

A + B = [
  1+5  2+6
  3+7  4+8
] = [
  6  8
  10  12
]

Matrix Subtraction

For matrices A and B of the same dimension, the difference A - B is a matrix C where cij = aij - bij for all i, j.

Example: Matrix Subtraction

Using the same matrices A and B as above, the difference A - B is:

A - B = [
  1-5  2-6
  3-7  4-8
] = [
  -4  -4
  -4  -4
]

Properties of Matrix Addition

  • Commutative: A + B = B + A
  • Associative: (A + B) + C = A + (B + C)
  • Identity: A + O = A, where O is the zero matrix (all elements are 0)
  • Additive Inverse: A + (-A) = O, where -A is the negative of A

Scalar Multiplication

Scalar multiplication involves multiplying each element of a matrix by a scalar value (a single number).

If k is a scalar and A is a matrix, then kA is a matrix where each element is k times the corresponding element in A: (kA)ij = k × aij.

Example: Scalar Multiplication

For the scalar k = 2 and the matrix:

A = [
  1  2  3
  4  5  6
]

The scalar product 2A is:

2A = [
  2×1  2×2  2×3
  2×4  2×5  2×6
] = [
  2  4  6
  8  10  12
]

Properties of Scalar Multiplication

  • Distributive over Matrix Addition: k(A + B) = kA + kB
  • Distributive over Scalar Addition: (k + m)A = kA + mA
  • Associative with Scalars: k(mA) = (km)A
  • Identity: 1A = A

Matrix Multiplication

Matrix multiplication is a binary operation that produces a matrix from two matrices. Unlike addition, the dimensions of the matrices must be compatible, not necessarily identical.

Compatibility for Multiplication

To multiply matrix A (m × n) by matrix B (p × q), the number of columns in A must equal the number of rows in B (n = p). The resulting matrix C will have dimensions m × q.

Calculation of Matrix Product

For the product C = AB, the element cij is calculated as the dot product of the i-th row of A and the j-th column of B:

cij = Σ(aik × bkj) for k = 1 to n
Example: Matrix Multiplication

For:

A = [
  1  2
  3  4
]

and

B = [
  5  6
  7  8
]

The product AB is:

AB = [
  (1×5 + 2×7)  (1×6 + 2×8)
  (3×5 + 4×7)  (3×6 + 4×8)
] = [
  19  22
  43  50
]

Properties of Matrix Multiplication

  • Not Commutative: Generally, AB ≠ BA
  • Associative: (AB)C = A(BC)
  • Distributive over Addition: A(B + C) = AB + AC and (A + B)C = AC + BC
  • Identity: AI = IA = A, where I is the identity matrix
  • Zero Property: A × O = O × A = O, where O is the zero matrix

The Identity Matrix

The identity matrix I is a square matrix with 1s on the main diagonal and 0s elsewhere. For any matrix A, AI = IA = A (when the dimensions allow multiplication).

Example: Identity Matrix

The 3 × 3 identity matrix is:

I3 = [
  1  0  0
  0  1  0
  0  0  1
]

Special Matrices and Operations

Transpose of a Matrix

The transpose of an m × n matrix A, denoted as AT, is an n × m matrix where rows and columns are interchanged: (AT)ij = aji.

Example: Matrix Transpose

For the matrix:

A = [
  1  2  3
  4  5  6
]

The transpose AT is:

AT = [
  1  4
  2  5
  3  6
]

Properties of Transpose

  • Double Transpose: (AT)T = A
  • Transpose of Sum: (A + B)T = AT + BT
  • Transpose of Product: (AB)T = BTAT
  • Transpose of Scalar Product: (kA)T = kAT

Symmetric and Skew-Symmetric Matrices

A square matrix A is:

  • Symmetric if A = AT
  • Skew-Symmetric if A = -AT
Example: Symmetric and Skew-Symmetric Matrices

A symmetric matrix:

S = [
  1  2  3
  2  4  5
  3  5  6
]

A skew-symmetric matrix:

K = [
  0  2  -1
  -2  0  3
  1  -3  0
]

Note that in a skew-symmetric matrix, the main diagonal must consist of zeros.

Triangular Matrices

  • Upper Triangular: All elements below the main diagonal are zero.
  • Lower Triangular: All elements above the main diagonal are zero.
  • Diagonal Matrix: All elements off the main diagonal are zero.
Example: Triangular Matrices

Upper triangular matrix:

U = [
  1  2  3
  0  4  5
  0  0  6
]

Lower triangular matrix:

L = [
  1  0  0
  2  4  0
  3  5  6
]

Diagonal matrix:

D = [
  1  0  0
  0  4  0
  0  0  6
]

Applications of Matrix Operations

Systems of Linear Equations

Matrices provide a compact way to represent and solve systems of linear equations. The system:

a11x1 + a12x2 + ... + a1nxn = b1
a21x1 + a22x2 + ... + a2nxn = b2
...
am1x1 + am2x2 + ... + amnxn = bm

Can be written as Ax = b, where A is the coefficient matrix, x is the variable vector, and b is the constant vector.

Example: Matrix Representation of Linear Systems

The system:

2x + y = 5
3x - 2y = 4

Can be written as:

[
  2  1
  3  -2
] [
  x
  y
] = [
  5
  4
]

Graph Theory and Adjacency Matrices

In graph theory, matrices can represent relationships between vertices in a graph. In an adjacency matrix A for a graph with n vertices, aij = 1 if there is an edge from vertex i to vertex j, and aij = 0 otherwise.

Computer Graphics and Transformations

In computer graphics, matrices are used to represent and apply transformations such as scaling, rotation, and translation. These operations can be composed by multiplying their respective transformation matrices.

Example: Rotation Matrix

A 2D rotation by angle θ is represented by the matrix:

R = [
  cos(θ)  -sin(θ)
  sin(θ)  cos(θ)
]

To rotate a point (x, y) by θ, we multiply:

[
  cos(θ)  -sin(θ)
  sin(θ)  cos(θ)
] [
  x
  y
] = [
  x·cos(θ) - y·sin(θ)
  x·sin(θ) + y·cos(θ)
]
