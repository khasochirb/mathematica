Vector Operations

Understanding vector addition, subtraction, and scalar multiplication

Introduction to Vectors

A vector is a mathematical object that has both magnitude (length) and direction. Unlike scalars, which are quantities described by a single number, vectors require multiple components to be fully specified. Vectors can represent various physical quantities such as displacement, velocity, acceleration, and force.

Vector Notation

There are several ways to denote vectors:

  • Bold letters: v, a, F
  • Letters with arrows: v⃗, a⃗, F⃗
  • Component form: v = (v₁, v₂, v₃) or v = v₁i + v₂j + v₃k

Types of Vectors

  • Zero vector (0): A vector with zero magnitude and undefined direction
  • Unit vector: A vector with magnitude 1
  • Parallel vectors: Vectors pointing in the same or opposite directions
  • Position vector: A vector from the origin to a specific point
Example 1: Position Vectors

Consider the points A(3, 4, 2) and B(1, -2, 5) in a three-dimensional space.

The position vector of A is:

OA = (3, 4, 2) = 3i + 4j + 2k

The position vector of B is:

OB = (1, -2, 5) = i - 2j + 5k

The vector from A to B is calculated as:

AB = OB - OA = (1, -2, 5) - (3, 4, 2) = (-2, -6, 3) = -2i - 6j + 3k

Vector Addition and Subtraction

Vector Addition

Vector addition combines two or more vectors into a resultant vector. If u = (u₁, u₂, u₃) and v = (v₁, v₂, v₃), then:

u + v = (u₁ + v₁, u₂ + v₂, u₃ + v₃)

Geometrically, vector addition follows the parallelogram law or the triangle law:

  • Parallelogram Law: The sum of two vectors is the diagonal of the parallelogram formed by the vectors
  • Triangle Law: When placed head-to-tail, the sum of two vectors is the vector from the tail of the first to the head of the second

Properties of Vector Addition

  • Commutative: u + v = v + u
  • Associative: (u + v) + w = u + (v + w)
  • Identity: v + 0 = v
  • Inverse: v + (-v) = 0

Vector Subtraction

Vector subtraction is defined as the addition of the negative of a vector:

u - v = u + (-v) = (u₁ - v₁, u₂ - v₂, u₃ - v₃)
Example 2: Vector Addition and Subtraction

Given vectors u = (3, -1, 4) and v = (2, 5, -2), find u + v and u - v.

Vector Addition:

u + v = (3, -1, 4) + (2, 5, -2) = (3+2, -1+5, 4+(-2)) = (5, 4, 2)

Vector Subtraction:

u - v = (3, -1, 4) - (2, 5, -2) = (3-2, -1-5, 4-(-2)) = (1, -6, 6)

Alternatively, we can express these in component form:

u + v = (3i - j + 4k) + (2i + 5j - 2k) = 5i + 4j + 2k

u - v = (3i - j + 4k) - (2i + 5j - 2k) = i - 6j + 6k

Scalar Multiplication

Scalar multiplication involves multiplying a vector by a scalar (a real number). If v = (v₁, v₂, v₃) and c is a scalar, then:

cv = (cv₁, cv₂, cv₃)

Geometrically, scalar multiplication:

  • Changes the magnitude of the vector by a factor of |c|
  • Maintains the direction if c > 0
  • Reverses the direction if c < 0
  • Results in the zero vector if c = 0

Properties of Scalar Multiplication

  • Distributive over vector addition: c(u + v) = cu + cv
  • Distributive over scalar addition: (c + d)v = cv + dv
  • Associative for scalar multiplication: c(dv) = (cd)v
  • Identity: 1v = v
Example 3: Scalar Multiplication

Given vector v = (3, -4, 2), find:

a) 2v

b) -0.5v

Solution:

a) 2v = 2(3, -4, 2) = (2×3, 2×(-4), 2×2) = (6, -8, 4)

b) -0.5v = -0.5(3, -4, 2) = (-0.5×3, -0.5×(-4), -0.5×2) = (-1.5, 2, -1)

Alternatively, in component form:

a) 2v = 2(3i - 4j + 2k) = 6i - 8j + 4k

b) -0.5v = -0.5(3i - 4j + 2k) = -1.5i + 2j - k

Linear Combinations of Vectors

A linear combination of vectors is formed by multiplying vectors by scalars and adding the results. Given vectors v₁, v₂, ..., vₙ and scalars c₁, c₂, ..., cₙ, their linear combination is:

c₁v₁ + c₂v₂ + ... + cₙvₙ

Span of Vectors

The span of a set of vectors is the set of all possible linear combinations of those vectors. For vectors in ℝ³:

  • The span of one non-zero vector is a line through the origin
  • The span of two non-parallel vectors is a plane through the origin
  • The span of three non-coplanar vectors is the entire ℝ³ space

Linear Independence

A set of vectors is linearly independent if none of the vectors can be expressed as a linear combination of the others. Mathematically, vectors v₁, v₂, ..., vₙ are linearly independent if the only solution to:

c₁v₁ + c₂v₂ + ... + cₙvₙ = 0

is c₁ = c₂ = ... = cₙ = 0.

Example 4: Linear Combinations and Independence

Consider the vectors u = (1, 2, 1), v = (3, 1, -1), and w = (7, 4, -1).

Part 1: Express w as a linear combination of u and v, if possible.

We need to find scalars a and b such that w = au + bv:

(7, 4, -1) = a(1, 2, 1) + b(3, 1, -1)

This gives us the system of equations:

a + 3b = 7

2a + b = 4

a - b = -1

Solving from the last two equations: 2a + b = 4 and a - b = -1

Adding these: 3a = 3, so a = 1

Substituting back: 1 - b = -1, so b = 2

Checking: 1 + 3(2) = 7 ✓

Therefore, w = 1u + 2v = u + 2v

Part 2: Are the vectors u, v, and w linearly independent?

Since we found that w = u + 2v, we can write:

u + 2v - w = 0

This is a non-trivial solution to a₁u + a₂v + a₃w = 0 (with a₁ = 1, a₂ = 2, a₃ = -1), so the vectors are linearly dependent.

Vector Magnitude and Unit Vectors

Vector Magnitude

The magnitude or length of a vector v = (v₁, v₂, v₃) is given by:

|v| = ||v|| = √(v₁² + v₂² + v₃²)

Unit Vectors

A unit vector has a magnitude of 1. To create a unit vector in the direction of v, divide v by its magnitude:

v̂ = v/|v|

The standard unit vectors in ℝ³ are:

  • i = (1, 0, 0): unit vector along the x-axis
  • j = (0, 1, 0): unit vector along the y-axis
  • k = (0, 0, 1): unit vector along the z-axis
Example 5: Magnitude and Unit Vectors

Given vector v = (3, -4, 12), find:

a) The magnitude of v

b) A unit vector in the direction of v

Solution:

a) |v| = √(3² + (-4)² + 12²) = √(9 + 16 + 144) = √169 = 13

b) The unit vector in the direction of v is:

v̂ = v/|v| = (3, -4, 12)/13 = (3/13, -4/13, 12/13)

or in component form: v̂ = (3/13)i - (4/13)j + (12/13)k

Applications of Vector Operations

Physics Applications

  • Displacement: The net displacement when moving along multiple path segments is the vector sum of the individual displacements
  • Force: The net force on an object is the vector sum of all forces acting on it
  • Velocity and Acceleration: Vector operations describe changes in motion and direction

Engineering Applications

  • Structural Analysis: Vectors represent forces and moments in structural elements
  • Electrical Engineering: Vectors describe electric and magnetic fields
  • Robotics: Vectors represent positions, velocities, and accelerations of robot joints and end effectors

Computer Graphics

  • Transformations: Vector operations enable rotations, translations, and scaling of objects
  • Lighting and Shading: Vectors represent surface normals and light directions
  • Animation: Vector interpolation creates smooth transitions between keyframes
Example 6: Physics Application

Three forces act on an object: F₁ = (2, 3, -1) N, F₂ = (-1, 2, 2) N, and F₃ = (3, -2, 4) N. Find the resultant force and its magnitude.

Solution:

The resultant force is the vector sum of all forces:

F_resultant = F₁ + F₂ + F₃ = (2, 3, -1) + (-1, 2, 2) + (3, -2, 4)

F_resultant = (2-1+3, 3+2-2, -1+2+4) = (4, 3, 5) N

The magnitude of the resultant force is:

|F_resultant| = √(4² + 3² + 5²) = √(16 + 9 + 25) = √50 ≈ 7.07 N

Therefore, the object experiences a resultant force of approximately 7.07 N in the direction of the vector (4, 3, 5).
